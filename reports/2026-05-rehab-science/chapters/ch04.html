04CLINICAL ASSESSMENT · 评估篇 / 第四章

怎么判断你伤到了哪里——SOAP 评估、特殊检查与影像分级 SOAP · Special Tests · Imaging Grades

康复的第一公里不是治疗,是评估。把疼痛、动作、力量、感觉、影像编织成一份「这个人发生了什么、为什么、接下来怎么走」的临床推理——这就是 SOAP 框架的全部使命。本章把它逐字段拆开:从 OPQRST 病史采集,到 MMT 肌力分级,再到 Lachman、McMurray、Neer 等 14 大特殊试验的敏感度/特异性表,再到 MRI 何时该开、何时是噪声,最后回到 PROM 量表与临床推理的整合。

▼ 三十秒看懂这一章

SOAP 是评估的骨架——Subjective(病史)负责「故事」,Objective(查体)负责「证据」,Assessment(评估)负责「推理」,Plan(计划)负责「行动」。临床检查不是影像的廉价替代,在很多场景里是金标准本身——Lachman 对前交叉韧带撕裂敏感度 85-87%、特异性 94%,与 MRI 同等甚至更优。盲目预约 MRI 既贵又可能制造「无关异常」造成患者焦虑——无症状人群腰椎 MRI 异常率 43%、肩袖部分撕裂 30%、半月板信号改变 20%。MMT 肌力 0-5 级 + ROM + 14 个特殊试验 + PROM 量表(KOOS、DASH、ODI、VAS)= 循证评估的金字塔。患者讲完故事时,你已经有 80% 的诊断。

故事 / 凯斯西储大学医学院 · 1968

1968 年,凯斯西储大学医学院,内科医生 Lawrence Weed 在《新英格兰医学杂志》(NEJM) 上发表了一篇看似不起眼的论文——《Medical Records that Guide and Teach》。在这之前,医生病历的形式是「时间流叙事」:今天病人主诉什么,医生写什么,杂乱无章地堆在一起。换一个值班医生,翻三页都不知道患者现在到底是什么问题。

Weed 提出一个革命性的想法:病历应该按问题组织,每个问题分四段——Subjective(主观),Objective(客观),Assessment(评估),Plan(计划)。问题清单 + 每个问题的 SOAP 进展,构成 Problem-Oriented Medical Record (POMR)。

这看上去像是一个文书规范,实际是一场认知革命。它强迫医生把「患者说什么」(主观)和「你看到什么」(客观)在同一份病历中并列——两条独立的证据线,任何一条不能单独决定结论;它强迫医生在做 Plan 之前先写一句「我认为这是 X」的明确 Assessment——你的推理过程必须可被审查;它强迫医生把「这个人」按「问题」拆解,而不是按「就诊次数」记录,因为一个人可能同时有 3 个问题在不同阶段。

SOAP 在 1970 年代被美国医学院普遍采用,1980 年代成为护理与物理治疗的通用语,如今几乎所有 EHR(电子病历系统)的默认模板都基于 POMR。对运动康复尤其关键——患者讲「跑步时膝盖外侧痛」(S),你查体看到 Ober 阳性、髋外展弱(O),你写下「考虑 ITBS,排除外侧半月板撕裂」(A),你计划下次教离心训练 + Ober 复测 + 髋外展进阶(P)——四段下来,任何接手的同事 60 秒能跟上你。

1976 年,加拿大骨科医生 John Lachman 在《临床骨科》上描述了一种简单的膝关节检查:仰卧屈膝 20-30°,一手稳大腿,一手前拉胫骨,感受胫骨向前移位的终末感(end feel)。他的结论是这个 30 秒的徒手动作对前交叉韧带撕裂的诊断,优于当时盛行的「前抽屉试验」(anterior drawer)。后续 Benjaminse 2006 系统综述证实——Lachman 敏感度 85-87%、特异性 94%,与 MRI 相当甚至更优。同期 Hoppenfeld 1976 的《Physical Examination of the Spine and Extremities》和 Cyriax 1978 的临床评估教科书,把这一代「徒手循证检查」固化成今天的标准——这就是本章的全部背景。

4.1SOAP 四段框架——Subjective Objective Assessment Plan

SOAP 不是文书规范,是临床思维的脚手架。一个完整的评估流程必须经过四个独立但有序的阶段:听患者讲故事(Subjective),做客观检查(Objective),做出推理判断(Assessment),写出可执行计划(Plan)。把这四段切开看,每一段都有自己独立的方法学、独立的常见错误、独立的训练要求。

S · Subjective · 主观(15-20 分钟)

这是患者「讲故事」的环节。包含主诉(chief complaint)、现病史(history of present illness, HPI)、既往史、家族史、社会史、用药史、过敏史。在运动康复语境中,HPI 是绝对核心——按 OPQRST(Onset / Provocation / Quality / Region / Severity / Timing)八轴展开,平均 15-20 分钟。临床上 70-90% 的诊断信息来自这一段(Hampton 1975 经典论文:门诊医生倾听病史得到 82% 的诊断信息,体检贡献 9%,实验室/影像贡献 9%)。

O · Objective · 客观(15-25 分钟)

这是治疗师「看证据」的环节。包含视诊 (inspection)、触诊 (palpation)、ROM 测量、肌力评定 (MMT)、特殊试验 (special tests)、神经学检查 (neurological exam)、功能筛查 (FMS / SFMA)、姿势评估等。每一项都必须用客观、可重复、可量化的方式记录——而不是「患者活动度还可以」。具体写「肩外展 AROM 145° (健侧 175°),终末感僵硬伴疼痛 5/10」。

A · Assessment · 评估(5-10 分钟)

这是推理输出。把 S 和 O 的信息综合,给出:① 工作诊断(working diagnosis)或鉴别诊断列表(differential dx);② ICF 框架下的功能障碍(身体结构/功能、活动、参与);③ 预后评估(prognosis)与目标;④ 排除红旗信号 (red flags)。这一段是 SOAP 里最考验功力的——它把零散的「证据」转化为「故事」。

P · Plan · 计划(5-10 分钟)

这是行动列表。包含:① 干预 (intervention) ——手法、运动疗法、电疗、教育等;② 转介 (referral) ——是否需要影像、骨科、神经科、营养咨询;③ 患者教育 (HEP, home exercise program);④ 复诊频率与时间表;⑤ 重新评估 (re-eval) 节点。Plan 必须 SMART——Specific、Measurable、Achievable、Relevant、Time-bound。

FIG 4.1 · SOAP 四象限——评估的认知脚手架,Weed 1968 提出后成为全球医疗记录的统一范式

SOAP 的设计哲学:S 和 O 的物理隔离。患者讲的不能和你看的混同——这是医学认识论的基础。把它们分开记录,后续接手的医生可以独立验证你的推理。如果 S 说「右膝外侧痛」但 O 找不到任何外侧体征,你需要重新审视——可能是牵涉痛?可能是患者表达问题?可能是你触诊位置不对?S 与 O 的不一致本身就是诊断线索。

原理:SOAP = 把诊断推理拆成 4 个互不污染的认知步骤。翻译:就像律师不能让自己的预判污染证据采集——你不能因为患者一进门主诉「跑者膝」就只查膝盖。S 听完先列鉴别清单(髌股 / ITBS / 髌腱 / 半月板 / 牵涉痛),O 段挨个用客观证据验证或排除,A 段写出「我为什么选 X 而不是 Y」,P 段才是「怎么治」。放在康复场景:一份合格的初评 SOAP 长度约 1500-2500 字,初次评估 45-60 分钟,复评 15-25 分钟。短于这个时长的「SOAP」通常是省略了 A 段——直接从 O 跳到 P,这是新手最常见的失误。

4.2病史采集 (S, Subjective)——OPQRST 八轴问诊

病史采集不是「问问题」,是建模——你在患者讲述的同时,在脑子里搭建出「这是什么伤、它怎么发生的、现在在哪一阶段、它对这个人意味着什么」的四维模型。问诊不是清单的机械执行,是结构化叙事的引导。下面把运动康复中最实用的 OPQRST 框架展开。

OPQRST 八轴

  • O — Onset(起病):什么时候开始?突然 (acute) 还是渐进 (insidious)?有明确受伤事件还是无?这是急性 vs 累积过载性损伤的第一分水岭——「我上周打球时膝盖一拧」(急性,考虑韧带/半月板)vs「跑步两个月以来膝外侧渐渐酸」(累积,考虑 ITBS/PFPS)。
  • P — Provocation / Palliation(加重/缓解):什么动作让它更痛?什么让它舒服?这是机械性疼痛的指纹——下楼痛 → PFPS;下坡跑痛 → ITBS;晨起头几步痛 → 足底筋膜炎/跟腱病;坐着抬腿痛 → 髌腱炎;仰卧抬起会痛、坐久了痛 → 椎间盘问题。
  • Q — Quality(性质):刺痛?钝痛?酸?胀?灼烧?麻木?「针刺感」「电击感」「火烧感」是神经根性疼痛的典型描述;「深部钝痛 + 边界不清」是肌肉/韧带性;「锐利定位明确」是结构性(骨/关节面)。
  • R — Region / Radiation(部位/放射):指给我看痛点。能用一根手指点出来吗?它会传到哪儿去?放射性疼痛常提示神经根受累或牵涉痛——颈痛放射到手指(C5-T1 神经根);腰痛放射到臀部以下(L4-S1);肩痛放射到手肘外侧(C5/C6 或肩袖牵涉)。
  • S — Severity(强度):用 0-10 评分(VAS / NPRS),目前是几?最痛时几?最不痛时几?问三个分数而非一个——单一「7 分」无法捕捉波动,但「现在 3、最差 8、最好 1」立刻显示出疼痛是断续性而非持续性。
  • T — Timing(时序):一天里什么时候最痛?跑步开头 vs 跑开后 vs 跑后第二天?夜间是否影响睡眠?夜间痛或休息痛是红旗信号(警惕应力骨折、严重炎症、肿瘤、感染等)。
  • + Aggravating Activities(影响功能):它让你做不了什么?跑步?上下楼?抱孩子?系鞋带?这是 ICF 框架中的「活动 / 参与」维度——比单纯「痛」更重要,因为它决定了康复目标。
  • + Patient Goals(目标):你希望恢复到什么程度?能慢跑 5K?能回到马拉松?能正常上下班?这一问从评估的开始就锚定了治疗的终点——不锚目标的康复是开放式赌博。

红旗筛查 (Red Flags)——病史里的雷

每次问诊都必须扫一遍下列「红旗」信号——它们提示可能不是单纯肌骨问题,需要医学转介:

  • 夜间痛 / 休息痛:警惕骨折、感染、肿瘤;
  • 不明原因体重下降(6 个月 >5%);
  • 持续发热;
  • 癌症既往史 + 新发肌骨痛(尤其腰椎、骨盆);
  • 大小便失禁或感觉异常(马尾综合征,腰椎红旗);
  • 双侧或进行性神经症状;
  • 近期严重外伤(高能机制伤);
  • 长期糖皮质激素使用(骨质疏松+骨折风险);
  • 静脉注射药物史 / 免疫抑制(感染风险);
  • 步态突变、平衡丧失(中枢/神经原因);
  • 女性闭经 ≥ 3 个月 + 任何骨痛(RED-S / 三联征);
  • 胸痛 / 心悸 / 晕厥(立即就医)。

原理:OPQRST 八轴 = 让每个症状都被多个独立维度交叉验证,避免单一描述造成误导。翻译:同一个「膝盖痛」,O 不同(急性扭伤 vs 跑步两个月)+ P 不同(下楼痛 vs 下蹲痛)+ R 不同(髌骨下 vs 膝外侧 vs 关节线)+ T 不同(晨起痛 vs 跑后痛)= 完全不同的诊断。放在康复场景:同一个患者主诉「膝痛」,如果按 OPQRST 八轴展开是 8 个独立信号——8 个里只要 3-4 个对上某个诊断的「指纹」,你的工作诊断就已经 80% 落定,后面 O 段只需要验证和排除。新手只问「哪儿痛、痛多久、有多痛」三轴 = 信息量不够,做不出鉴别。

问诊技巧 · 开放式问 + 沉默

临床问诊的两个核心技巧:① 开放式问题先于封闭式——「描述一下你的疼痛」比「是不是刺痛?」好;Beckman 1984 经典研究发现:平均 18 秒后医生会打断患者——这一打断丢掉了 23% 的关键信息;② 沉默是工具——患者讲完一段后,如果你不急着问下一个问题、保持 3-5 秒沉默,患者往往会自己补充最重要的那句话——「其实……我还有一件事没说」。试试看。

4.3视诊 (Inspection)——姿势、肿胀、瘀斑、对称性

视诊是 O 段的第一帧——患者一进门、脱了外套、坐下来,你已经在看了。视诊不需要患者配合,但需要你系统、有意识、左右对照。一份合格的视诊涵盖姿势、对称性、皮肤、肿胀、肌肉萎缩、步态、辅助器具使用——所有信息都用「客观语言」记录。

静态视诊清单

  • 整体姿势(站立位):头前倾?圆肩驼背?骨盆前/后倾?脊柱侧弯?膝过伸/屈曲?足旋前/旋后?(参考第 3 章姿势评估)
  • 左右对称性:两侧肩高、髂嵴高、膝高、踝高;两侧肌肉体积(尤其股四头肌、臀大肌、小腿三头肌);两侧手臂自然摆放角度。
  • 皮肤:发红 (erythema)、瘀斑 (ecchymosis)、皮温升高、出汗异常、营养性改变(色素沉着、毛发脱落)、伤疤、手术切口。
  • 肿胀(swelling/effusion):局部还是弥漫?急性(几小时内)还是慢性?膝关节看髌上囊和髌韧带两侧凹陷是否消失;踝关节看外踝凹陷;手指看指节是否「腊肠样」。
  • 肌肉萎缩:左右对照测量周径(膝上 10 cm 测股四头肌;踝上 15 cm 测小腿三头肌);差异 >1 cm 视为有意义。
  • 畸形:任何明显的骨性、关节性畸形(脱位、半脱位、骨突);皮下结节(类风湿结节、痛风石、腱鞘囊肿)。

动态视诊——步态分析

让患者走 10 米,前后两次。看的是:

  • 对称性:左右步长相等吗?摆臂对称吗?
  • 步速:正常 1.2-1.4 m/s;<1.0 m/s 提示明显功能受损。
  • 跛行(antalgic gait):一侧承重时间缩短 = 该侧疼痛或不稳。
  • Trendelenburg 步态:站立相骨盆向对侧倾斜 = 站立侧臀中肌弱(髋外展弱)。
  • 跨阈步态(steppage):足下垂、膝抬高过度 = 腓总神经损伤、L5 神经根。
  • 剪刀步态:两膝交叉 = 内收肌张力过高(脑瘫、中风后)。
  • 蹒跚步态(waddling):骨盆左右大幅摇摆 = 双侧髋外展弱(肌营养不良、髋关节炎)。
  • 共济失调步态(ataxic):宽基底、不稳、不能直线行走 = 小脑或后索病变。

触发记录的实操语言

不要写「患者膝盖有点肿」——写「右膝髌上囊及髌韧带两侧凹陷消失,sweep test 阳性,1+ 积液,皮温较对侧轻度升高 (~1°C),无红斑」。后者是可重复的客观数据,任何接手的人都能理解,2 周后复查可比较。

4.4触诊 (Palpation)——结构与功能的接触

触诊是评估中最被低估的技能。一双经过训练的手能在 30 秒内分辨出肌肉、韧带、肌腱、关节囊、骨突的层次,识别压痛点、肌张力、皮温、肿胀边界、组织顺应性、可触及的解剖标志。视诊看「形态」,触诊摸「质感」。

触诊的系统化层次

Hoppenfeld 1976 的经典教科书 Physical Examination of the Spine and Extremities 提出触诊的分层系统——从浅到深,逐层评估:

  1. 皮肤:温度、湿度、纹理、移动度(粘连?)、皮下结节;
  2. 皮下脂肪与浅筋膜:厚度、压痛、肿胀;
  3. 肌肉:张力、压痛、触发点 (trigger point)、左右对照、抗阻收缩时的形态;
  4. 肌腱:走行、压痛、增厚、捻发音 (crepitus);
  5. 韧带与关节囊:压痛、肿胀、缺损;
  6. 骨与骨突:压痛、畸形、骨表面突起;
  7. 关节:关节线压痛、积液波动、关节面捻发音。

关键解剖标志的触诊

下面是康复评估中必触的核心标志(以下肢为例):

  • 髂前上棘 ASIS:骨盆前倾/后倾的对照点。
  • 髂后上棘 PSIS:骨盆旋转、骶髂关节评估的锚点。
  • 大转子:髋关节高度对照、转子滑囊炎压痛点。
  • 髌骨上下极:髌腱止点(下极)、股四头肌腱止点(上极);髌腱炎在下极压痛。
  • 关节线(膝):屈膝 90° 时膝中线两侧的「沟」即关节线,半月板撕裂时关节线压痛。
  • 胫骨结节:髌腱止点,Osgood-Schlatter 病的压痛点。
  • 腓骨头:外侧副韧带止点、腓总神经走行点。
  • 内/外踝:踝韧带定位的基准。
  • 跟腱中段(止点上 2-6 cm):中段跟腱病的触诊带——增厚、压痛、捻发音都在这里。
  • 足底筋膜跟骨止点:足底筋膜炎的主要压痛点。
  • 第五跖骨基底:Jones 骨折部位,踝扭伤鉴别必触。

压痛分级

压痛的标准临床分级(Magee 2014):

  • 0:无压痛;
  • I 度:患者主诉痛;
  • II 度:患者皱眉;
  • III 度:患者退缩 (withdraw);
  • IV 度:不允许触及。

原理:触诊 = 用手作为「分辨率高于影像的传感器」分层读取组织信息。翻译:训练有素的手能感知到 MRI 看不到的——肌张力高低、皮下粘连、轻度积液、组织滑动性。放在康复场景:跟腱中段触诊找到「梭形增厚 + 局部压痛 + 抗阻提踵时增厚段紧绷」三联征 = 教科书中段跟腱病;ITBS 触诊找到「股骨外侧髁上方 2-3 cm 处局部压痛 + 屈伸膝时压痛加重」= 教科书 ITBS;McMurray 阳性 + 内侧关节线压痛 = 内侧半月板撕裂高度怀疑。触诊不是 O 段的「附加项」,是 O 段的核心信息源——它在你写下「Lachman 阴性」之前,就已经决定了你接下来要做哪些试验。

反共识 / 触诊比 MRI 更敏感(在很多场景里)

影像看「结构」,触诊摸「功能」——MRI 能拍到跟腱中段梭形增厚,但 MRI 无法回答「这个增厚段是炎症期还是退变稳定期、是不是真的痛、抗阻收缩时是否紧绷」。这些都得靠手摸出来。新手往往认为「先做个 MRI 看看」,但 80% 的肌骨问题在熟练触诊 + 病史 + 简单特殊试验之后,就已经不需要 MRI 了——MRI 只是排除严重病变 + 手术规划的工具,不是诊断的起点。

4.5ROM 测量——主动 vs 被动活动度,关节囊型态评估

关节活动度 (Range of Motion, ROM) 是所有肌骨评估的基础量化指标。它衡量关节在三维空间中可达的范围,反映关节囊、韧带、肌肉、肌腱的协同状态。ROM 不是越大越好——它必须对称、无痛、终末感正常。下面把 AROM、PROM 与关节囊型态评估讲透。

AROM vs PROM

AROM (Active ROM):患者自己主动完成的活动范围。涉及肌肉力量、神经控制、关节通畅性、疼痛禁制——是功能性的测量。
PROM (Passive ROM):治疗师被动完成的活动范围(患者放松)。只受关节囊、韧带、骨性结构限制——是结构性的测量。

PROM 通常比 AROM 大 5-15°(因为去除了主动收缩的能量需求和保护性禁制)。这两个数字的差值本身就是诊断线索:

  • AROM = PROM 且都受限:关节囊/韧带/骨性问题(粘连、骨刺、关节囊紧缩);
  • AROM < PROM 显著:肌力不足、神经损伤、疼痛禁制、肌腱断裂;
  • AROM = PROM 且都正常:本节点无问题。

终末感 (End Feel)——关节囊型态的诊断

这是 Cyriax 1978 提出的关键概念:被动推到 ROM 终点时,治疗师的手感知到的「阻挡的质感」。每种正常关节都有典型终末感,异常终末感是诊断线索。

关节囊型态 (Capsular Pattern)

当关节囊整体发炎或纤维化时(如冻结肩、髋关节炎),关节活动受限呈特定方向比例——这就是关节囊型态。Cyriax 的经典型态:

  • 肩:外旋 > 外展 > 内旋(冻结肩典型);
  • 髋:屈曲、内旋、外展受限明显(髋关节炎);
  • 膝:屈曲 >> 伸展(膝关节炎,屈曲首先受限);
  • 踝:跖屈 > 背屈(踝关节炎);
  • 颈椎:旋转和侧弯各方向均受限。

4.6肌力评定 0-5 级 (MMT, Manual Muscle Testing)

徒手肌力测试 (Manual Muscle Testing, MMT) 是 Henry Kendall & Florence Kendall 1949 提出、Daniels & Worthingham 1986 完善的 6 级量表(0-5),是康复评估中最通用的肌力评估方法。它不需要任何设备,30 秒内可测一块肌肉,是治疗师必备的核心技能。

0-5 级量表

级别名称定义临床意义

4良好 Good抗中等阻力完成全 ROM,但弱于健侧有功能但需训练

每级还可加 + / – 进一步细分(如 4+ 比 4 更接近 5,4– 比 4 略弱)。4 级是临床最常见的「异常」——既有功能又有缺陷,绝大多数康复目标设在「从 4 到 5」。

MMT 的局限性——以及如何弥补

MMT 是有用的快速筛查,但有明显局限:

  • 天花板效应:对运动员尤其明显——一个能深蹲 150 kg 的跑者股四头肌「MMT 5 级」毫无意义,因为测试者的手不可能比 150 kg 强;
  • 主观性大:不同测试者评分的kappa 系数仅 0.4-0.7,信度中等;
  • 对耐力无评估:只测瞬间最大力量,不测肌肉耐力 (endurance);
  • 对单关节孤立肌群较准,对多关节复合动作不适合。

弥补方法:① Handheld Dynamometer 手持测力计——量化到牛顿值,信度提升至 kappa 0.8-0.9;② 1-RM 测试(深蹲、硬拉、卧推)——运动员场景的金标准;③ 等速肌力测试(Biodex / Cybex)——研究场景的金标准;④ 计次测试——单腿桥到力竭次数、提踵到力竭次数,左右对照。

关键肌群的 MMT 测试位

  • 髋外展(臀中肌):侧卧,上侧腿伸直外展,治疗师在大腿外侧施加向下阻力;
  • 髋伸展(臀大肌):俯卧屈膝 90°,抬大腿,阻力在大腿后方;
  • 股四头肌:坐位屈膝 90°,伸膝,阻力在小腿前;
  • 腘绳肌:俯卧屈膝 45°,继续屈膝,阻力在小腿后;
  • 踝背屈(胫前肌):坐位,踝背屈,阻力在足背;
  • 踝跖屈(小腿三头肌):站立单腿提踵,正常 ≥25 次;<25 次提示弱;
  • 腓骨长短肌:坐位,踝外翻,阻力在足外侧;
  • 肩外展(三角肌中束):坐位,肩外展 90°,阻力在上臂外侧;
  • 肩外旋(冈下肌):侧卧或贴墙,屈肘 90°,前臂外旋,阻力在腕背侧;
  • 颈深屈肌:仰卧,下巴内收(craniocervical flexion),不能用胸锁乳突肌代偿。

原理:MMT = 用治疗师标准化的徒手阻力,把肌力分到 6 个粗粒度的功能区间。翻译:它不能告诉你「股四头肌是 78 kg 还是 85 kg」,但能告诉你「这块肌肉处在什么功能阶段,下一步训练目标是什么」。放在康复场景:ACL 重建术后第 8 周,股四头肌从 MMT 2 进步到 4 = 可以进入闭链负重训练(深蹲、登台阶);MMT 4 进步到 5(左右对称且抗最大阻力)= 可以开始跑步回归。MMT 不是「测肌力」,是「测康复阶段」——它是阶段晋级的决策工具。

运动员场景 · MMT 之外

对运动员,MMT 5 几乎是「废话级别」的天花板。运动员需要的是功能性肌力测试:① 单腿提踵到力竭次数(健侧 30+,患侧 ≤ 80% 即弱);② 单腿深蹲质量(能否到 60° 而不内扣);③ 等速肌力 H/Q 比(腘绳/股四,正常 60-70%,<55% 增加腘绳伤风险);④ 跳跃测试(单腿跳距离 LSI,患/健 < 90% 不可回归运动);⑤ 1RM 比值(健侧深蹲 100 kg 患侧 70 kg = LSI 70%,不达标)。

4.7特殊试验 (Special Tests)——膝关节 6 大试验

特殊试验是 O 段的诊断核心——每个试验都针对一个特定结构或病变,有自己的敏感度 (sensitivity)、特异性 (specificity)、阳性预测值 (PPV)、阴性预测值 (NPV)。下面把膝关节最常用的 6 大试验讲透,数据来自 Benjaminse 2006、Hegedus 2007、Décary 2017 等系统综述。

1 · Lachman 试验 · ACL 前交叉韧带

方法:仰卧屈膝 20-30°,治疗师一手稳定股骨远端,另一手握胫骨近端向前拉。感受胫骨向前移位的距离与终末感 (end feel)。

阳性:胫骨向前移位 >3 mm 较健侧、或软终末感(无硬挡)。

循证(Benjaminse 2006 系统综述):敏感度 85-87%、特异性 94%。这是膝关节最敏感的徒手试验之一,与 MRI 相当甚至更优。麻醉下测试敏感度可达 95% 以上。

临床意义:ACL 撕裂诊断的金标准徒手试验。Lachman 阴性几乎可排除 ACL 完全撕裂;阳性 + 急性血肿 = 95% 是 ACL 撕裂。

2 · 前抽屉试验 (Anterior Drawer) · ACL

方法:仰卧屈膝 90°,治疗师坐在患者足上,双手握住胫骨近端,向前拉。

阳性:胫骨前移 >6 mm。

循证:敏感度 仅 55%、特异性 92%。显著低于 Lachman——急性损伤后腘绳肌保护性紧张可掩盖前移,且 90° 屈膝时关节面几何也减少前移幅度。

临床意义:Lachman 是首选;前抽屉作为补充。新手最常见错误是「我用了前抽屉,阴性所以 ACL 没事」——错。

3 · McMurray 试验 · 半月板

方法:仰卧屈膝最大,治疗师一手握足、一手按膝关节线。胫骨外旋 + 缓慢伸膝(查内侧半月板);胫骨内旋 + 缓慢伸膝(查外侧半月板)。

阳性:关节线触及「咔嗒」声、或患者感觉关节线疼痛、或闪缩反应。

循证(Hegedus 2007 综述):敏感度 61%、特异性 84%。单独使用诊断力中等,需要与关节线压痛 (joint line tenderness) 联合——后者敏感度 83%、特异性 83%。两者联合诊断半月板撕裂的整体准确度可达 80% 以上。

临床意义:McMurray + 关节线压痛 + Apley 三联,是半月板撕裂的徒手诊断组合。

4 · Apley 试验 · 半月板

方法:俯卧屈膝 90°,治疗师一手稳大腿,另一手沿胫骨长轴向下压并旋转(压挤试验);然后向上拉并旋转(分离试验)。

阳性:压挤旋转时痛 → 半月板;分离旋转时痛 → 韧带。

循证:敏感度 22-60%、特异性 70-90%。敏感度差异较大,作为辅助试验。

5 · Valgus / Varus 应力试验 · 内/外侧副韧带

方法:仰卧屈膝 30°,治疗师一手压膝外侧、另一手扳踝向外(查 MCL 内侧副韧带);反向操作查 LCL 外侧副韧带。

阳性:关节内侧/外侧开口 >5 mm 较健侧,或终末感软。

循证:敏感度 86-96%、特异性 85-100%。MCL 撕裂诊断的金标准徒手试验。

临床意义:30° 屈曲位测纯韧带损伤;0° 完全伸直位测试如果阳性 = 不仅 MCL/LCL,还有 ACL/PCL/关节囊联合损伤,通常更严重。

6 · 髌骨研磨试验 (Patellar Grind / Clarke『s Sign) · PFPS

方法:仰卧伸膝,治疗师手压髌骨向远端,要求患者主动股四头肌收缩。

阳性:髌骨下疼痛或股四收缩中断。

循证:敏感度 49%、特异性 75%。诊断价值有限,假阳率高(很多无症状者也会引出疼痛)。

临床意义:不能单独诊断 PFPS,需要结合「下楼痛 + 蹲痛 + 久坐痛 (movie sign) 」的病史 + 髌骨外推恐惧 (apprehension test) 等多个体征。

新手最容易犯的错:看到一个 +Lachman 就写「ACL 撕裂」。错。没有一个特殊试验能单独诊断疾病——所有诊断都是临床推理的综合输出。Lachman 阳性 + 急性扭伤史 + 即时血肿 + 不能继续比赛 + 听到「啪」一声 = 95% 是 ACL 撕裂。Lachman 阳性但无外伤史 + 无血肿 = 可能是慢性松弛、个体差异、操作误差。特殊试验是证据的一票,不是判决。

4.8特殊试验——肩关节 + 踝关节 8 大试验

肩关节和踝关节是骨科门诊和运动康复中第二、第三高频部位(膝第一)。下面把这两个关节最实用的 8 大特殊试验讲透。

肩关节 5 大试验

1 · Neer 试验 · 肩峰下撞击:治疗师固定肩胛骨,被动肩前屈至最大。阳性:肩前外侧痛 (40-80°)。循证:敏感度 72%、特异性 60%(Hegedus 2008)。

2 · Hawkins-Kennedy 试验 · 肩峰下撞击:屈肘 90°、肩前屈 90°,被动内旋。阳性:肩前外侧痛。循证:敏感度 79%、特异性 59%。Neer + Hawkins 联合阴性可基本排除肩峰下撞击。

3 · 空罐试验 (Empty Can) · 冈上肌:肩外展 90° + 水平内收 30° + 拇指向下(模拟「空罐」),抗阻向上举。阳性:疼痛或无力。循证:敏感度 74%、特异性 61%。无力比疼痛更预测撕裂。

4 · 跌落试验 (Drop Arm) · 大型肩袖撕裂:被动肩外展 90°,要求患者缓慢放下手臂。阳性:手臂在 90° 附近突然「跌落」或不能控制下降。循证:敏感度 27%、特异性 97%。低敏感高特异——阳性几乎确诊大撕裂,阴性不能排除。

5 · O『Brien 试验 (Active Compression) · SLAP / AC 关节:肩前屈 90° + 水平内收 15° + 拇指向下,抗阻向下压;然后拇指向上重复。阳性:拇指向下时痛,向上时缓解。循证:敏感度 67%、特异性 37%(SLAP 撕裂)。

踝关节 3 大试验

6 · 前抽屉试验 (Anterior Drawer) · ATFL 前距腓韧带:踝跖屈 10-20°,一手稳胫骨远端,另一手握跟骨向前拉。阳性:前移 >5 mm 较健侧或软终末感。循证:敏感度 74%、特异性 38%(急性),敏感度 96%、特异性 84%(>5 天后,因为急性肿胀降低敏感度)。所以踝扭伤要在 4-5 天后复查——急性期前抽屉的诊断价值低。

7 · 距骨倾斜试验 (Talar Tilt) · CFL 跟腓韧带:踝中立位,一手稳胫骨,另一手内翻足。阳性:角度 >10° 较健侧或软终末感。循证:敏感度 52%、特异性 84%(>5 天后)。

8 · Thompson 试验 · 跟腱完全断裂:俯卧屈膝 90°,治疗师挤压小腿三头肌。阳性:足无跖屈反应 = 跟腱完全断裂。循证:敏感度 96%、特异性 93%。跟腱断裂的金标准徒手试验,30 秒搞定——比 MRI 都快。

4.9神经学检查——肌力反射感觉、神经张力试验

神经学检查 (neurological exam) 是 O 段中最少被新手认真做、但又是最关键的一段——一个被忽略的L5 神经根受压会让所有「腰椎康复」变成无效操作。神经学检查的四大支柱:肌力 (MMT)、深腱反射 (DTR)、感觉 (sensory)、神经张力试验 (neural tension)。

1 · 肌力 · 神经根代表肌

每个神经根都有典型「代表肌」(myotome)——这块肌肉的弱化提示对应神经根受累:

2 · 深腱反射 (DTR, Deep Tendon Reflex)

反射弧的完整性测试。叩击肌腱,观察反射收缩。0-4+ 分级:

  • 0:无反射(下运动神经元损伤);
  • 1+:反射减弱(下神经元损伤、肌肉萎缩);
  • 2+:正常;
  • 3+:反射亢进(上运动神经元损伤、焦虑);
  • 4+:阵挛(明确上神经元损伤,脊髓损伤、中风、ALS)。

常用反射:① 肱二头肌反射 C5-6;② 肱三头肌反射 C7;③ 髌反射 L3-4;④ 跟腱反射 S1。Babinski 征(足底刺激后大拇趾背伸 + 其他趾扇形展开)= 上运动神经元损伤的红旗,正常成人应为屈曲反应。

3 · 感觉 · 皮节图 (Dermatomes)

每个神经根支配一个皮节区域。用棉签或大头针测轻触觉 / 痛觉,沿皮节分布检查。常用解剖标志:

  • C5:三角肌外侧
  • C6:拇指、食指
  • C7:中指
  • C8:小指、无名指
  • T1:前臂内侧
  • T4:乳头水平
  • T10:脐水平
  • L1:腹股沟
  • L2-3:大腿前内侧
  • L4:小腿内侧、内踝
  • L5:小腿外侧、足背、大拇趾
  • S1:足外侧缘、小趾、足底外侧
  • S2-4:会阴区、肛周(必查于腰痛红旗筛查!)

4 · 神经张力试验 (Neurodynamic Tests)

这是 David Butler 1991 系统提出的神经组织滑动评估——神经从中枢到末梢需要在椎管和组织间隙中自由滑动,任何卡压、粘连、炎症都会引起神经被牵拉时的疼痛。

SLR (Straight Leg Raise) 直腿抬高试验 · L4-S1 神经根

方法:仰卧,治疗师抬起患者直腿,记录引起疼痛的角度。
阳性:30-70° 间出现典型放射性下肢痛 (radicular pain),沿坐骨神经分布。
循证:敏感度 91%、特异性 26%。SLR 阴性几乎排除腰椎间盘突出(SnNout);但阳性需要进一步评估。
交叉 SLR(抬健侧腿引起患侧痛):敏感度仅 29% 但特异性 88% — 高特异性提示中央型大突出。

Slump 试验 · 神经根整体张力

方法:坐位,患者依次:① 弓背 (slump);② 颈前屈;③ 伸单膝;④ 踝背屈。每一步后询问症状变化。
阳性:任何一步引出沿神经分布的放射性痛。逐步释放(如颈伸展后疼痛消失)= 强阳性(神经源性而非肌肉源性)。
循证:敏感度 84%、特异性 83%。比 SLR 更全面,神经张力评估的首选。

ULTT (Upper Limb Tension Test) · 上肢神经张力

评估颈神经根(C5-T1)沿臂丛的张力。有 4 个变体,分别针对正中神经、桡神经、尺神经。阳性:沿神经分布的放射痛或感觉异常,通常在 90° 外展 + 外旋 + 伸肘 + 腕背伸时出现。
循证:敏感度 72-97%、特异性 33-87%(根据变体不同)。颈椎神经根病的高敏感度筛查工具。

红旗 / 马尾综合征 (Cauda Equina Syndrome)

这是腰椎康复中绝对不能漏的红旗——L1 以下椎管内多个神经根受压。表现:① 鞍区(会阴/肛周)麻木;② 大小便失禁或潴留;③ 双侧下肢神经症状;④ 性功能突然丧失。一旦出现,必须 24-48 小时内手术减压,延误超过 48 小时永久性损伤风险骤升。任何腰痛患者都必须问这四个问题——「最近大小便有没有变化?会阴区有没有麻木?」患者可能羞于主动说,你必须问。

4.10功能性动作筛查 FMS / SFMA / Y-Balance

特殊试验和 ROM 测的是「孤立的零件」,功能筛查测的是「整车的运动模式」。Gray Cook 2010 提出的功能性动作筛查 (Functional Movement Screen, FMS) 把全身基本动作模式拆成 7 个测试,每项 0-3 分,总分 21 分。后续 SFMA(选择性功能评估)和 Y-Balance Test 进一步细化。

FMS 7 项测试

  1. 深蹲 (Deep Squat):双脚与肩同宽,木棒举过头,深蹲到底。评估髋膝踝肩同步活动。
  2. 跨栏步 (Hurdle Step):单腿跨越障碍。评估单腿稳定 + 髋屈曲 + 踝背屈。
  3. 直线弓步 (In-line Lunge):前后脚一条线弓步。评估髋分离活动 + 核心稳定。
  4. 肩部活动 (Shoulder Mobility):双手背后相互触摸。评估肩复合体活动度。
  5. 主动直腿抬高 (Active SLR):仰卧直腿抬高。评估腘绳活动 + 髋分离。
  6. 俯卧撑稳定 (Trunk Stability Push-up):俯卧撑姿势 + 起身。评估核心抗伸展稳定。
  7. 旋转稳定 (Rotary Stability):四点支撑,对侧手脚抬起。评估核心抗旋转稳定。

每项评分:

  • 3 分:无代偿完成,完美完成;
  • 2 分:有代偿但能完成;
  • 1 分:不能完成;
  • 0 分:动作中出现疼痛(无论分数,该项目降为 0 + 必须查疼痛来源)。

FMS 的循证现实

FMS 在 2010 年代被广泛推广,声称「FMS ≤ 14 分预测损伤风险升高 2-3 倍」(Kiesel 2007)。但后续多个高质量综述(Bonazza 2017、Moran 2017 等)的结论更谨慎:

  • 低分组确实损伤率略高,但预测价值有限(灵敏度 ~40%);
  • 不同测试者评分一致性:训练有素者 kappa ~0.6-0.8,新手 ~0.4-0.5;
  • 缺乏对运动员特异性的考虑——FMS 不是为高水平运动员设计的;
  • 分数提高是否减少损伤:证据弱。

结论:FMS 是有用的动作模式筛查(找出明显代偿和不对称),但不是有力的损伤预测工具。把它当作筛查的起点,不是终点。

SFMA · 选择性功能评估

SFMA 是 Cook 团队 2014 后开发的临床版 FMS——专为有症状者设计(FMS 为无症状者设计)。从 10 个 top-tier 动作开始,任何「功能性疼痛」(FP) 或「非功能性疼痛」(DN/DP) 都通过下钻分解到具体的关节、肌肉、稳定性问题。这是临床评估的强大补充,特别适合非特定性腰痛、肩痛等模糊主诉。

Y-Balance Test · 动态平衡评估

Plisky 2006 提出。患者单腿站立,另一脚分别向前、后内、后外三个方向尽量伸出,测量距离。三个方向距离归一化(/ 下肢长度)后总和。

循证:

  • 左右不对称 >4 cm:下肢损伤风险升高 2.5 倍(Plisky 2006,高中运动员);
  • 归一化总和 <94%:下肢损伤风险升高(具体阈值因人群而异);
  • ACL 重建术后:Y-Balance 是回归运动的关键测试之一,LSI 应 >90%。
放在你身上 · 30 分钟自助筛查清单

你 35 岁、办公室坐 10 小时/天、想恢复跑步,但担心「身体没准备好」。给自己做一次结构化筛查:

  1. 姿势(5 min):对着镜子看头前倾度、圆肩、骨盆前倾;墙站测试(后脑-后背-臀部贴墙,看下背腰部能否塞进一只手——能塞进 = 正常,塞不进或能塞 2 只手都不对)。
  2. ROM 自测(8 min):① 髋屈曲(仰卧抱膝到胸 ≥110°);② 髋伸展(俯卧抬腿 ≥15°);③ 踝背屈(墙距测试,膝触墙最远脚距 ≥10 cm);④ 肩外旋(贴墙背手摸到对侧肩胛骨);⑤ 颈侧弯(耳到肩 ≥40°)。
  3. 力量自测(10 min):① 单腿提踵到力竭(健侧 ≥25 次,双侧差异 <20%);② 单腿桥到力竭(≥20 次);③ 平板支撑(≥60 秒);④ 单腿深蹲(能否到 60° 而不内扣);⑤ 北欧腘绳(下放控制 ≥30° 屈髋)。
  4. 平衡(5 min):单脚闭眼站(健康成人 ≥30 秒)。
  5. 动作模式(2 min):深蹲到底(双脚跟不离地、膝不内扣、躯干前倾不超 30°);弓步前后(膝不超过脚尖 + 后膝触地)。

结果解读:① 任一项 ROM 比对侧差 >10° → 需针对性活动度训练;② 任一力量项达不到指标 → 重点强化(单腿桥不到 20 次 = 髋后链弱);③ 单脚闭眼站 < 15 秒 = 本体感觉差,跑步前先做平衡训练;④ 深蹲膝内扣 + 跟离地 = 踝背屈不足 + 髋外展弱,跑步前一定要解决。30 分钟,你已经做了一份比大多数 PT 第一次评估更详细的自评。

4.11影像金标准与陷阱——X 光 / CT / MRI / 超声

影像是康复评估中最被滥用的工具——「先去拍个 MRI」是最常见的偷懒决策。每种影像模态有自己的适用、局限、辐射、成本。下面把四大模态讲透,然后讲一个反常识:无症状者的 MRI 异常率惊人——影像本身可能制造焦虑。

1 · X 光 (Plain Radiograph)

适用:① 骨折、脱位(急诊首选);② 退行性改变(关节炎、骨刺);③ 骨结构异常(畸形、肿瘤、感染晚期);④ 钙化(钙化性肌腱炎、Achilles 跟骨突)。
不适用:软组织(韧带、肌腱、半月板、椎间盘)X 光看不到。
辐射:胸片 ~0.1 mSv(等同 10 天背景辐射);四肢 X 光 <0.01 mSv,极低。
成本:低 (中国 30-200 元,美国 100-400 美元)。
局限:软组织无法评估;应力骨折早期 X 光阴性(2-3 周后才出现骨痂);不能用 X 光排除应力骨折。

2 · CT (Computed Tomography)

适用:① 复杂骨折(三维评估手术方案);② 椎间盘 / 椎管骨性结构;③ 急性头部 / 胸腹外伤;④ 关节内游离体定位。
不适用:软组织对比度低于 MRI;非急诊场景下不优于 MRI。
辐射:胸 CT ~7 mSv,腹盆 CT ~10 mSv = 一年背景辐射的 3-5 倍。反复 CT 有累积致癌风险,儿童尤甚。
成本:中(中国 300-800 元,美国 500-1500 美元)。

3 · MRI (Magnetic Resonance Imaging)

适用:① 韧带 / 肌腱撕裂 (ACL、肩袖、跟腱、半月板);② 椎间盘突出;③ 应力骨折(MRI 是金标准,早期 1 周内即可显示骨髓水肿);④ 软组织肿块;⑤ 关节软骨评估。
不适用:① 急诊创伤(优先 X 光/CT 排骨折);② 体内有铁磁性植入物 (老式心脏起搏器、动脉瘤夹);③ 严重幽闭恐惧未做镇静。
辐射:0 mSv,无电离辐射。
成本:高 (中国 700-2000 元,美国 1500-3500 美元)。
序列:T1 看解剖;T2 看水肿/液体(亮);PD 看半月板;STIR 看骨髓水肿(应力骨折)。

4 · 超声 (Diagnostic Ultrasound)

适用:① 浅层肌腱(跟腱、髌腱、肩袖)实时动态评估;② 关节积液;③ 神经卡压(腕管、肘管);④ 引导注射 (PRP、激素);⑤ 床旁评估,即时反馈。
不适用:骨内结构(超声无法穿透骨);深部结构(髋深部、椎间盘);肥胖患者图像质量下降。
辐射:0 mSv,无电离辐射。
成本:低-中(中国 100-500 元,美国 250-700 美元)。
独特优势:动态评估——能在收缩/活动中看肌腱滑动、撞击;床旁实时——可在诊室同时做诊断和评估治疗效果(如注射后即时复查)。

无症状者的 MRI 异常率——影像的「假阳性」陷阱

这是影像的关键反共识:无症状的健康人群,MRI 异常率惊人。多项大型流行病学研究证实:

  • 腰椎间盘突出(无症状者):20 岁人群 ~30%、40 岁 ~50%、60 岁 ~65%、80 岁 ~84%(Brinjikji 2015 系统综述);
  • 颈椎间盘异常(无症状者):50 岁以上 ~80% 有椎间盘退变;
  • 肩袖部分撕裂(无症状者):50 岁以上 ~30%,70 岁以上 ~60%(Sher 1995);
  • 半月板撕裂信号(无症状者 >50 岁):~25-35%(Englund 2008);
  • 髋关节盂唇撕裂(无症状者):~70%(Register 2012);
  • 跟腱异常信号(无症状跑者):~30%。

这意味着:如果你给一个 50 岁的无症状跑者做全身 MRI,你一定会找到「异常」——但这些异常和他/她的疼痛未必相关。把这些异常告诉患者(「您椎间盘突出了!您半月板撕裂了!」),会引发灾难化认知,严重者出现「影像伪致残」(imaging-induced disability)——患者因为知道自己「有伤」而停止活动,最终真的失能。

原理:影像异常 ≠ 临床症状。无症状人群异常率极高,影像只能在「临床高度怀疑某诊断」的前提下做验证。翻译:影像不是体检,是验证工具。先临床评估「我认为这是 X」,再用影像「确认 X」——而不是先做影像「看看有什么」。放在康复场景:一个跑者主诉膝外侧痛,Ober + 阳性 + 髋外展弱 + 下坡跑加重 = 临床诊断 ITBS,根本不需要 MRI;一个 ACL 撕裂高度怀疑(Lachman + 急性血肿)= MRI 是为术前规划 + 排除合并伤(MCL、半月板、骨挫伤)而非「确诊 ACL」;一个非特异性慢性腰痛(无放射、无神经症状、无红旗)= 不需要 MRI,MRI 找到的「异常」反而可能延缓康复进程。

反共识 / MRI 制造焦虑 · 影像伪致残

Jensen 1994 NEJM 经典研究:对 98 名无症状志愿者做腰椎 MRI——52% 有椎间盘膨出,27% 突出,1% 脱出。Carragee 2006 进一步证实:这些「异常」对未来疼痛和功能的预测力几乎为零——它们就像皱纹一样,是身体老化的正常现象,不是「病」。但患者一旦被告知「您腰椎有问题」,行为开始改变:不敢弯腰、不敢跑步、不敢搬东西。这种「知道异常导致行为改变,行为改变导致失能」的级联反应,被命名为「影像伪致残」。循证康复的临床原则:能不做的影像不做;必须做的,要给患者讲清楚「异常 ≠ 病」。

实操 / 何时该开 MRI · 五条决策规则
  1. 临床高度怀疑严重病变需手术或排除红旗(ACL 撕裂术前规划、应力骨折、肿瘤、感染、马尾综合征);
  2. 保守治疗 4-6 周无效 + 影响生活功能(顽固性肩痛、慢性腰痛对治疗反应差);
  3. 临床检查不能确定鉴别诊断(同时怀疑 ACL + 半月板 + MCL 多结构损伤);
  4. 需要客观证据指导手术决策(肩袖撕裂大小、椎间盘突出位置);
  5. 专业运动员的回归运动决策(高规格证据要求)。

不要做 MRI 的情况:① 急性踝扭伤(优先 Ottawa 决策规则 + X 光);② 无症状跑者「想体检看看」;③ 急性腰痛 <6 周 + 无红旗(85% 自愈);④ 单纯肌肉酸痛;⑤ 任何只为「让自己放心」的影像。

4.12PROM 量表——KOOS、DASH、ODI、VAS、NPRS

PROM (Patient-Reported Outcome Measures, 患者报告的结局指标) 是主观评估的循证工具——把患者的「感觉怎么样、能做什么、生活质量如何」用标准化问卷量化。PROM 是 SOAP 的 S 段最有力的补充,也是康复疗效追踪的金标准。

1 · 疼痛量表 · VAS 与 NPRS

VAS (Visual Analogue Scale, 视觉模拟评分):10 cm 直线,左端「无痛」,右端「极痛」,患者标记位置,测量 cm 数。范围 0-10。
NPRS (Numerical Pain Rating Scale, 数字疼痛评分):0-10 自报,「0 = 无痛,10 = 最痛」。临床更常用,简单口头沟通即可。
MCID(最小临床重要差异):通常 2 分。即「从 7 → 5」才算有意义的改善,「从 7 → 6」属于测量误差范围。
实操技巧:问 3 个值——「目前」「最痛时」「最不痛时」,捕捉波动。

2 · 膝关节 · KOOS / IKDC

KOOS (Knee Injury and Osteoarthritis Outcome Score):42 项问卷,5 个子量表——疼痛、症状、ADL、运动/休闲、QoL。每子量表 0-100 分,100 = 完美。
MCID:各子量表 8-10 分。
IKDC (International Knee Documentation Committee):18 项,适合年轻活动人群(ACL 术后)。
临床应用:ACL 重建术后随访金标准。

3 · 上肢 · DASH / Quick-DASH

DASH (Disabilities of Arm, Shoulder and Hand):30 项,评估手臂、肩、手整体功能(肩袖损伤、腕管综合征、肱骨外上髁炎等)。
Quick-DASH:11 项简化版,2-3 分钟完成,信效度与全版相当。
MCID:~10 分。
临床应用:肩肘腕康复随访;手术 vs 保守治疗效果比较。

4 · 腰背 · ODI / Roland-Morris

ODI (Oswestry Disability Index):10 项,每项 0-5 分,转换百分比。慢性腰痛功能评估的金标准。
分级:0-20% 轻度;20-40% 中度;40-60% 重度;60-80% 严重残疾;>80% 卧床。
MCID:10 分。
Roland-Morris (RMDQ):24 项,适合急性腰痛;简短易答。

5 · 髋 · HOOS / iHOT

HOOS (Hip disability and Osteoarthritis Outcome Score):KOOS 髋版本,40 项。
iHOT-33 / iHOT-12:专为年轻髋关节患者(髋撞击综合征 FAI、盂唇撕裂)设计。

6 · 足踝 · FAAM / AOFAS

FAAM (Foot and Ankle Ability Measure):29 项,2 个子量表(ADL 21 项 + 运动 8 项)。
AOFAS:美国足踝外科学会评分,医生填写为主(包含客观体征)。

7 · 通用功能 · LEFS / SF-36 / EQ-5D

LEFS (Lower Extremity Functional Scale):20 项,任何下肢损伤通用。
SF-36 / SF-12:健康生活质量通用量表;
EQ-5D:5 维度健康通用量表,卫生经济学常用。

PROM 的临床应用 · 五个要点

  • ① 初评 + 复评同一量表——纵向比较才有意义;
  • ② MCID 是判读关键——「分数提高 1」不算改善,必须达到 MCID;
  • ③ 选择部位特异性量表 > 通用量表(KOOS > SF-36 for ACL);
  • ④ 加入心理量表筛查——慢性痛 30-50% 合并抑郁焦虑,影响康复结局(FABQ、PHQ-9);
  • ⑤ 数字化 + 自动化——让患者用手机填,治疗师只看趋势图,效率高 10 倍。
PROM 选择决策树

① 疼痛 = VAS/NPRS(每次都要);② 部位特异 + 患者活动水平:年轻运动员膝伤 → IKDC;中老年膝关节炎 → KOOS;急性腰痛 → RMDQ;慢性腰痛 → ODI;肩袖术后 → Quick-DASH 或 Constant Score;③ + 心理量表:疼痛 >3 个月、避痛行为明显、患者主诉「人生都变了」 → 加 FABQ + PHQ-9;④ 研究/正式报告:加 EQ-5D(卫生经济学)+ GRC(Global Rating of Change)。

4.13临床推理 (Clinical Reasoning)——把信息编织成决策

这是 A 段(Assessment)的全部内容,也是 SOAP 里最难、最不可外包给问卷的一段。临床推理不是「按清单打勾」,是把零散的证据编织成「这个人发生了什么」的连贯故事。下面把临床推理的认知科学讲透。

临床推理的双系统模型

Croskerry 2009 把临床推理拆成两个互补系统(基于 Kahneman 的双过程论):

  • 系统 1 · 模式识别 (Pattern Recognition):快速、自动、依赖经验。资深 PT 看 5 秒患者步态就「知道」是 Trendelenburg。这是经验内化后的「直觉」——准确性高,但易受偏倚影响。
  • 系统 2 · 假设演绎 (Hypothetico-Deductive):慢、刻意、基于规则。新手或复杂病例必用——「列出鉴别清单 → 用病史/查体逐项验证 → 排除/降序」。准确性高,但耗时。

专家级临床医师两个系统流畅切换——简单病例用系统 1 快速决策,复杂或异常病例切到系统 2 仔细推理。新手则必须几乎全程用系统 2。

鉴别诊断的「金字塔」

对任何主诉,临床推理都应该形成降序排列的鉴别诊断列表——从「最可能」到「不能漏」:

  1. 最可能(common):基于流行病学和病史最可能的诊断;
  2. 次可能:其他需要考虑的肌骨问题;
  3. 不能漏(can『t miss):虽然概率低但漏掉后果严重(骨折、肿瘤、感染、马尾、心源性、动脉夹层等)。

例:30 岁跑者「左小腿胫骨内缘酸痛 3 周」:

  • 最可能:MTSS(内侧胫骨应力综合征);
  • 次可能:胫骨应力反应早期、胫骨后肌腱炎、慢性筋膜室综合征;
  • 不能漏:胫骨应力骨折(尤其女性 LEA / 闭经者)、深静脉血栓(单侧肿胀 + 红温)、骨肿瘤(夜间痛、年轻、罕见但严重)。

临床检查的目的是升降这个列表——任何阳性体征要么让某诊断上升,要么把它从列表排除。

临床推理的常见偏倚 · 警戒清单

  • 锚定偏倚 (Anchoring):被第一个想法或第一个测试结果「锚住」,后续证据被选择性解读以支持它。解药:每次评估结束后问「如果我的工作诊断是错的,可能是什么?」
  • 可得性偏倚 (Availability):最近见过的、印象深的诊断更容易被想起。解药:定期回看罕见但重要的鉴别清单。
  • 确认偏倚 (Confirmation):只关注支持自己诊断的证据,忽略反对的。解药:主动列出「3 个反对当前诊断的证据」。
  • 过早闭合 (Premature Closure):在证据不充分时过早锁定诊断。解药:除非很有把握,初评不要给确定性 100% 的诊断,给「工作诊断 + 鉴别清单」。
  • 归因偏倚 (Attribution):把症状归因于患者「老了/胖/不锻炼」,而忽略具体病变。解药:对每个症状都问「假如这个人是 25 岁运动员,我会怎么想?」
  • 沉没成本 (Sunk Cost):已经开了一些治疗,即使无效也不愿改诊断。解药:4-6 周治疗无效就主动重新评估,不要继续无效干预。

ICF 框架 · 把诊断翻译成功能

WHO 的 ICF (International Classification of Functioning, Disability and Health) 是康复评估的核心概念框架——把医学诊断翻译成功能性影响:

  • 身体结构/功能 (Body structures/functions):解剖结构和生理功能;例:ACL 撕裂、股四头肌弱 MMT 3+、膝屈曲 ROM 90°。
  • 活动 (Activity):个人执行任务的能力;例:不能跑步、上下楼需扶栏、不能单腿站立 10 秒。
  • 参与 (Participation):在社会环境中的角色;例:不能继续每周 3 次球队训练、被迫缺席比赛、社交孤立。
  • 环境因素:家庭支持、工作要求、文化、物理障碍;
  • 个人因素:年龄、教育、生活方式、心理特质。

合格的 Assessment 段应该三个维度都涉及——结构层(ACL 撕裂)+ 活动层(不能跑步、不能单腿跳)+ 参与层(影响球队、心理压力)。只写「ACL 撕裂」是不够的——那只是疾病标签,不是康复目标。

推理输出模板

合格的 A 段应该包含:① 工作诊断 + 信心程度(「我认为是 X(70% 信心),次诊断 Y(20%),不能排除 Z(10%)」);② 主要功能限制 (ICF):结构 / 活动 / 参与三层;③ 预后(典型恢复时长 + 个体因素);④ 需要额外评估的方面(影像?转介?);⑤ 红旗状态(已排除 X、Y、Z 严重病变)。新手最缺的不是知识,是把推理过程写下来的习惯——一旦写下来,逻辑漏洞就立刻暴露。

把前 13 节的内容浓缩成一份可直接使用的临床评估流程图。从患者进门到 SOAP 完成,一个 45-60 分钟的初评应该按下列顺序展开。

0-5 min · 接诊与开场

  • 简短自我介绍 + 解释流程(降低患者焦虑);
  • 请患者用一句话说主诉:「您主要想解决什么?」;
  • 记录基本人口学(年龄、性别、职业、运动史)。

5-25 min · S · Subjective

  • HPI · OPQRST 八轴展开;
  • 既往史 PMH、用药 Meds、过敏 Allergy、家族史 FHx、社会史 SHx;
  • 红旗筛查(夜间痛、体重下降、发热、肠膀胱症状等);
  • 患者目标(回到什么程度?何时?);
  • PROM 初评(VAS + 部位特异量表)——可由患者在等候时填好。

25-50 min · O · Objective

  • 视诊:整体姿势、对称性、肿胀、瘀斑、皮肤改变、步态(让患者走 10 m);
  • 触诊:系统分层、关键标志、压痛点定位;
  • ROM:AROM + PROM(主要受累关节 + 上下游关节);记录终末感;
  • MMT:0-5 级,关键肌群(运动员场景用功能性测试补充);
  • 特殊试验:基于 S 段假设有针对性地选,不要全部跑一遍;
  • 神经学检查:任何放射性症状都必须做(MMT + DTR + 感觉 + 神经张力);
  • 功能筛查(可选):FMS/SFMA/Y-Balance 用于运动员或动作模式问题;
  • 影像复查:已有影像评估并记录关键发现。

50-60 min · A + P · Assessment + Plan

  • A · 评估:工作诊断 + 鉴别 + ICF 三层 + 预后 + 红旗状态;
  • P · 计划:
  • · 干预:手法 / 运动疗法 / 教育 / 电疗 / 其他;
  • · HEP 家庭训练(具体动作 + 次数 + 频率);
  • · 复诊频率(典型 2x/周 × 4-8 周);
  • · 重新评估节点(2-4 周复评关键指标);
  • · 转介(如需要 MRI、骨科、营养、心理);
  • · SMART 目标(2 周、6 周、12 周);
  • 患者教育:用 1-2 分钟告诉患者「您发生了什么、为什么、我们要做什么、需要多久」——共同决策 (shared decision-making) 是循证康复的核心。
放在你身上 · 一份完整 SOAP 病历示范

下面是一份针对前述「35 岁女性、跑步外侧膝痛 3 周」的合格 SOAP 病历:

S · Subjective:

  • 主诉:右膝外侧偏上痛 3 周,影响跑步训练
  • O · Onset:渐进起病,无外伤;3 周前周跑量从 25 km 加到 40 km;
  • P · Provocation:下坡跑明显加重,平路 5-10 min 后开始痛;下楼梯轻度痛;
  • Q · Quality:针刺样痛,定位精确;
  • R · Region:右膝外侧偏上 2-3 cm,1 指可点出;无放射;
  • S · Severity:NPRS 现在 1,最痛时 6,最不痛时 0;
  • T · Timing:跑中突然出现,跑后 1-2 h 消退;无夜间痛;
  • 既往史:无;月经规律;无 NSAID 长期使用;
  • 红旗:无(无夜间痛、无体重下降、无神经症状、无外伤);
  • 患者目标:6 周内回到 40 km/周稳定训练,8 周后参加半马训练营;
  • PROM:KOOS-Pain 78/100、KOOS-Sport 60/100。

O · Objective:

  • 视诊:站立位无明显畸形;右膝外侧轻度发红,无肿胀;步态无明显跛行;
  • 触诊:股骨外侧髁上 2-3 cm 处局部压痛 II 度,直径 ~3 cm;髂胫束远端紧张;无关节线压痛;
  • ROM:右膝屈伸 0-135° 全 AROM,无痛终末感正常;
  • MMT:右髋外展 4-(臀中肌)/ 5(健侧),右股四头肌 5,右腘绳肌 5;
  • 特殊试验:Ober + 阳性(髋内收受限至中线上 5°);Noble 阳性(屈伸膝 30° 时压痛复现);Lachman 阴性;McMurray 阴性;关节线压痛阴性;髌骨研磨阴性;
  • 神经学:DTR 正常,感觉正常,SLR 阴性;
  • 功能:单腿桥右 12 次 / 左 22 次,LSI 55%(显著弱);单腿深蹲右侧膝内扣 + 骨盆下沉;
  • 步态分析(短跑录像):触地时右膝 valgus + 髋内收过度,过度跨步 (overstride),步频 162。

A · Assessment:

  • 工作诊断:右髂胫束综合征 (ITBS)(85% 信心)——基于 Ober + Noble + 髂胫束远端压痛 + 典型病史(跑量陡增 + 下坡加重 + 定位精确);
  • 鉴别诊断:外侧半月板撕裂(已排除,关节线压痛阴性 + McMurray 阴性,5% 残留)、外侧副韧带应力反应(<5%)、髌股关节疼痛(<5%);
  • 不能漏:股骨颈应力骨折(无相关体征,排除);
  • 主要功能障碍:
  • · 结构:髂胫束远端撞击 + 髋外展弱 LSI 55%;
  • · 活动:不能 >10 min 跑步,下坡跑无法完成;
  • · 参与:训练计划中断,无法完成下周半马训练营准备课;
  • 预后:典型 ITBS 4-8 周保守治疗有效率 90%+,患者无慢性化风险,预后好;
  • 红旗:已排除(无夜间痛、无神经症状、无系统性表现)。

P · Plan:

  • 干预 · 本次:
  • · 患者教育(10 min):解释 ITBS 机制(挤压而非摩擦);跑量陡增是触发因素;现代康复重点在髋外展力量而非伸展;预期 4-8 周回归;
  • · 手法:髂胫束远端松动(MT),股骨外侧髁上局部按摩;
  • · HEP:① 减跑量至 15 km/周 × 2 周,平路为主,不下坡;② 单腿桥 3×15、侧卧蚌式 3×20、Pallof press 3×10/侧、单腿罗马尼亚硬拉 3×10/侧,每日;③ 步频暂时不调,2 周后开始 165 → 175 渐进;
  • 转介:暂无;
  • 复诊:1x/周 × 4 周,然后每 2 周一次;
  • 2 周目标:Ober - 距离改善 5°,单腿桥右 ≥ 18 次,跑量 25 km 无痛;
  • 6 周目标:KOOS-Sport ≥ 80,LSI ≥ 85%,跑量回到 40 km/周 无痛;
  • 8 周目标:完全回归训练,Ober 阴性 / 弱阳性,下坡跑 30 min 无痛;
  • 重新评估节点:2、4、6、8 周;
  • 如 4 周无改善:重新评估(MRI 排除其他诊断、运动医学转介)。

这就是一份完整的初评 SOAP——约 1800 字,45 分钟,可被任何接手的同事在 5 分钟内跟上你的推理。你也可以用它和患者沟通:展示给患者看,告诉他「我看到了什么,我怎么想,我们要做什么」——这是循证沟通的实质。

▼ 本章核心要点
  • SOAP(Subjective / Objective / Assessment / Plan)是临床评估的认知脚手架——S 和 O 物理隔离,A 必须显式推理,P 必须 SMART。
  • 诊断信息 70-90% 来自病史(Hampton 1975),不是影像;OPQRST 八轴问诊是结构化叙事的核心。
  • 视诊 + 触诊在很多场景比 MRI 更敏感——「梭形增厚 + 抗阻紧绷」摸出跟腱病;ITBS、PFPS、MTSS 都可临床诊断。
  • AROM ≠ PROM——两者的差异本身是诊断线索;终末感(end feel)是关节囊型态评估的核心。
  • MMT 0-5 级是康复阶段的决策工具,不是肌力的精确测量——运动员场景需用功能性测试补充。
  • Lachman 对 ACL 撕裂敏感度 85-87% / 特异性 94%,与 MRI 等同;Thompson 对跟腱断裂 96/93,30 秒搞定。
  • SnNout(高敏感阴性可排除)+ SpPin(高特异阳性可确诊)是临床流行病学的两个核心口诀。
  • 神经学检查不能省——肌力代表肌 + DTR + 皮节 + 神经张力试验(SLR、Slump、ULTT)。
  • FMS / SFMA / Y-Balance 是动作模式筛查,不是有力的损伤预测工具;LSI < 90% 不可回归运动。
  • 影像不是金标准,临床检查是——无症状者腰椎间盘异常率 30-65%、肩袖部分撕裂 30%、半月板信号 25-35%。
  • 能不做的 MRI 不做——「影像伪致残」是真实现象(Jensen 1994、Carragee 2006)。
  • PROM 量表(KOOS、DASH、ODI、VAS、FAAM)是 PR 主观评估的循证补充;判读关键是 MCID,不是分数本身。
  • 临床推理是双系统——模式识别(系统 1)+ 假设演绎(系统 2);避免锚定、可得性、确认、过早闭合等偏倚。
  • ICF 框架把医学诊断翻译成功能影响——结构 / 活动 / 参与三层缺一不可。
  • 一份合格初评 SOAP 约 1500-2500 字、45-60 分钟;复评 15-25 分钟。短于这个时长通常省略了 A 段(推理)。
PROTOCOL · 本章关键操作清单

协议 1 · 45 分钟初评 SOAP 时间分配

  1. 测什么: 一次完整的肌骨初评是否覆盖了 SOAP 全部四段 + ICF 三层 + 红旗排除
  2. 工具: SOAP 模板(纸质或电子)+ 量角器 + 手持测力计(可选) + 计时器
  3. 步骤(60 min 总):
    • 0-5 min: 接诊 + 主诉 + 患者目标
    • 5-25 min: S 段 · OPQRST 八轴 + 红旗筛查 + PROM 量表初评
    • 25-35 min: O 段 · 视诊 + 触诊 + 步态分析
    • 35-45 min: O 段 · ROM + MMT + 特殊试验(基于 S 段假设有针对性选)
    • 45-50 min: O 段 · 神经学检查(任何放射症状必做)+ 功能筛查(选做)
    • 50-55 min: A 段 · 工作诊断 + 鉴别 + ICF 三层 + 红旗状态
    • 55-60 min: P 段 · 干预 + HEP + SMART 目标 + 复诊频率 + 患者教育
  4. 频率: 每位新患者初评 1 次;复评 15-25 min × 每 2-4 周或重要节点
  5. 判读:
    • SOAP 总长 1500-2500 字 = 合格
    • A 段超过 100 字 + 三层 ICF + 鉴别列表 = 合格推理
    • P 段必须 SMART(具体、可测、可达、相关、有时限)
SOAP 中没有明确「红旗排除」记录 + 患者有放射症状或夜间痛 + 系统性表现 → 必须重新评估并加做神经学检查、考虑医学转介

协议 2 · 红旗筛查(每次评估必做)

  1. 测什么: 任何一项红旗存在 → 立即医学转介,不要在 PT 范围内处理
  2. 工具: 一张红旗清单 + 问诊技巧
  3. 问诊清单(逐项必问):
    • ① 夜间痛 / 休息痛(警惕骨折、肿瘤、感染)
    • ② 不明原因体重下降 6 月内 > 5%(肿瘤、系统性疾病)
    • ③ 持续发热 / 寒战(感染)
    • ④ 癌症既往史 + 新发肌骨痛(转移瘤)
    • ⑤ 大小便失禁 / 鞍区麻木(马尾综合征,腰椎红旗)
    • ⑥ 双侧或进行性神经症状(脊髓病、肿瘤)
    • ⑦ 近期严重外伤 + 高能机制(骨折、内脏伤)
    • ⑧ 长期糖皮质激素 / 免疫抑制(骨折、感染风险)
    • ⑨ 静脉吸毒史(感染性关节炎、脓肿)
    • ⑩ 步态突变 / 平衡丧失(中枢)
    • ⑪ 女性闭经 > 3 个月 + 任何骨痛(RED-S + 应力骨折)
    • ⑫ 胸痛 / 心悸 / 晕厥(心源性,立即急诊)
  4. 频率: 每次评估开始时扫一遍 + 病情变化时重扫
  5. 判读:
    • 无任何红旗 → 继续标准评估流程
    • 1 项红旗 → 详细询问 + 文件记录,考虑医学转介
    • 多项红旗 / 严重红旗(马尾、胸痛、休息夜间痛 + 体重下降)→ 当日急诊转介,不要继续 PT 评估
出现任何下列情况 → 立即急诊,不要「再观察一下」:① 鞍区麻木 + 大小便失禁(马尾综合征,48 小时内手术减压);② 胸痛 / 严重心悸 / 晕厥;③ 急性双侧神经症状 + 步态突然改变;④ 严重外伤 + 关节畸形 + 不能负重

协议 3 · 影像决策规则 · 何时该开 MRI

  1. 测什么: 是否需要影像;不必要的影像浪费金钱、增加焦虑、可能制造「影像伪致残」
  2. 工具: 一份影像决策规则清单 + 临床判断
  3. 开 MRI 的 5 条规则(必须满足任一):
    • ① 临床高度怀疑严重病变需手术或排除红旗(ACL 撕裂、应力骨折、肿瘤、感染、马尾)
    • ② 保守治疗 4-6 周无效 + 影响生活功能
    • ③ 临床检查不能确定鉴别诊断(同时怀疑 ACL + 半月板 + MCL 多结构)
    • ④ 需要客观证据指导手术决策(肩袖撕裂大小、椎间盘突出位置)
    • ⑤ 专业运动员的回归运动决策
  4. 不开 MRI 的情况:
    • ① 急性踝扭伤(优先 Ottawa 决策规则 + X 光)
    • ② 无症状跑者「想体检看看」
    • ③ 急性腰痛 < 6 周 + 无红旗(85% 自愈)
    • ④ 单纯肌肉酸痛 / DOMS
    • ⑤ 任何只为「让自己放心」的影像
  5. 开 X 光的情况: 急诊创伤 + Ottawa 规则阳性、疑似骨折、关节脱位、应力骨折后期(2-3 周后骨痂)
  6. 开超声的情况: 浅层肌腱动态评估、关节积液、引导注射、床旁即时反馈
  7. 判读 / 沟通模板:
    • 解释影像异常 ≠ 病:「您这个椎间盘膨出在您这个年龄段约 50% 的人都有,大多无症状」
    • 聚焦临床:「影像证实了我的临床判断,治疗方案不变」
    • 避免诊断标签致残:不要把「您椎间盘突出 / 您半月板撕裂」作为唯一沟通信息
影像异常但临床无对应症状 → 可能是「无症状异常」(50 岁腰椎 50% / 肩袖 30%),不要在没有临床推理支持下基于影像开始治疗或手术 → 患者应该治疗自己的症状,不是治疗自己的 MRI
