05SHELTER · 四要素 / 第五章

宿营——五避选址 · 庇护所 · 睡眠系统 · 失温预防 Shelter & Sleep

山里夜间的杀手不是猛兽不是悬崖,是温度和地面。一个营地的好坏在白天看不出来——白天天好,什么坡都能搭;晚上下雨起风,营地选址的每一个错误都会被放大五倍。这一章把「扎营」拆成四件事:选哪里(五避)、用什么搭(帐篷或原始庇护所)、地垫和睡袋怎么叠(睡眠系统)、为什么失温会在不冷的温度下杀人(湿冷午后悖论)。最后一节给一个最坏剧本:你没有任何装备,要靠地形和身体撑过一夜。

▼ 三十秒看懂这一章

宿营这件事的反共识有两条。第一条:失温最危险的温度不是 -10°C,是 5-10°C 配雨 + 风——前者人主动加衣,后者人觉得「还好」于是淋着走,体表散热在湿衣 + 风冷下放大十倍,核心温度悄悄掉到 32°C 才被发现。第二条:地垫比睡袋更被低估——地传导是夜间最大单一热损失通道,没有地垫的睡袋等于打了对折,R3+ 是三季最低线、R5+ 是雪季最低线。再加一条选址的硬规则:五避——避水、避风、避坠物、避兽道、避死木。这一章给你一份「5°C 配小雨 / 队友说休息一下」的剧本,以及最坏情况下没有装备一夜的庇护所方案。

故事 / 鳌太 · 2017 年 5 月 1 日午后

2017 年 5 月 1 日下午两点,陕西鳌山-太白纵穿主脊海拔 3400 米。当天气温实测 4 到 9°C,有零星小雨,风速 6-7 级。从塘口上山的二十多人队伍走到水窝子营地附近——这是鳌太线上海拔最高、最暴露的一段石海。队伍里很多人冲锋衣已经被汗水和雨水共同浸透,内层抓绒沉甸甸贴在身上。有一个三十多岁的男性,体能不错,前一晚营地他和大家一起喝过酒,早上起得比较晚,出发时穿得偏少。下午两点半左右,他开始走得明显慢,讲话节奏变快但内容颠倒,队友以为他只是累了,说休息一下就好。

这位先生在下午四点二十分左右停下不再走。他的体征记录后来在事故复盘里有详细描述:寒颤已经停止——这不是好转,这是中度失温(核心温度 32-35°C)的信号,身体把寒颤这个产热机制关掉了,因为肌肉的能量储备已经低到不足以维持寒颤。同伴试图用睡袋裹住他,但他的内层衣物全是湿的,睡袋一接触湿衣立刻失去保暖能力。两个小时后他失去意识。山里没有救援队能在这个时间点上到 3400 米。第二天上午救援抵达时,这是当年鳌太五月一日那场风波里多名遇难者之一。

那一天的实测温度是 4 到 9°C。不是 -20°C,不是高海拔暴风雪,是一个城里人会穿短袖出门的温度。但配上小雨 + 大风 + 内层湿透 + 走得疲劳 + 出发时穿得不够,这个温度区间在主脊上构成了最完美的失温杀手组合——这个组合在国外山岳救援文献里有个专门的名字,叫 湿冷午后(wet-cold afternoon)。它的危险不在数字,在心理:在这个温度下,人不警觉。

这一章后面所有的具体动作——选址五避、地垫 R 值、睡袋温标怎么看、失温分级处置——都是为了让你在看上去不冷的午后,不会重复 2017 年 5 月 1 日鳌太那位先生的剧本。

5.1选址五避——一片好坡地下面藏着五种地雷

白天看到一片平整的坡地、靠水方便、风景也好,大多数新手会立刻「就这里吧」。这是营地选址最常见的错误模式——只看正面,不查反面。一个夜间安全的营地必须主动通过五项排除:避水、避风、避坠物、避兽道、避死木。任何一项失守,白天看上去再好的坡也是一个等着被引爆的雷。

反共识 / 靠近水源不是优点,是减分项

新手最常犯的错:看到溪流就觉得「水源近真好」——直接扎在岸边 3-5 米。这是中国低海拔山区夜间溺亡事故的主要原因之一。任何一条山溪在 2-4 小时雷雨后都可以涨 0.5-2 米,泥石流来临时是30 秒的距离。正确做法是营地高于河滩历史最高水痕至少 2 米垂直高度、距离水源 30-50 米水平距离——打水多走一段路是用时间买安全。同样反共识的:也不要扎在「看上去干燥的河滩石头堆」——那里大概率是历史上洪水冲下来的石头,扎在上面等于扎在下次洪水的路径上。

5.2帐篷类型与搭建顺序——三季 / 四季 / Tarp 怎么选

帐篷不是「越贵越好」也不是「越大越好」——是「匹配场景」。中国温带山区三季多日穿越的主流选择是双层三季帐篷(1.5-2.5kg),雪山或冬季高海拔才上四季帐(3-4kg),夏季低海拔林区可选 Tarp + 蚊帐组合(0.5-1kg)。错配场景的最常见后果:三季帐顶不住雪季的风荷载、四季帐在三季的闷热雨夜结一身冷凝水。

搭建顺序——五步,新手最常错地钉角度

帐篷搭建是个有正确顺序的事——不是随便插哪个先都行。下面的五步顺序是 MSR / Big Agnes 等品牌说明书加 NOLS 教学的综合版,任何双层三季帐都适用。

  2. 地布(footprint)铺设——略小于帐底 5-10cm,绝对不能超出帐底外沿(否则雨水会沿地布流到帐内)

5.3原始庇护所——没有帐篷怎么办

帐篷丢了、被风撕了、或者你本来就是想体验「少装备」——这时候靠地形和本地材料造庇护所是基本功。主流五种:A 字框架、Lean-to 单坡棚、雪洞、树根坑、岩穴。每一种对应不同地形和材料,共同原则只有一个:保住三件事——挡风、挡雨、阻断地传导。这三件事做不到的庇护所等于露宿。

三件事的优先级——挡风 > 阻断地传导 > 挡雨

在时间和材料有限时,这三件事的优先级不是一样的。挡风排第一——风冷效应(wind chill)在 5°C 配 6 级风时等效温度 -7°C,失温速度翻倍;阻断地传导排第二——地面是夜间最大单一热损失通道(下一节详谈),铺一层厚 15-20cm 的干树枝 / 干草 / 松针等于一张临时地垫;挡雨排第三——除非已经在下雨,挡雨的优先级低于前两项,因为人体对短时间淋湿的耐受比对持续风冷高得多。

背后的机制——热损失四通道

这三件事的优先级不是凭直觉排出来的,是热力学决定的。人体散热只有四种方式:传导(conduction · 直接接触冷物,主要是地面)、对流(convection · 流动的空气和水,即风冷和湿冷)、辐射(radiation · 向冷的天空和环境辐射红外)、蒸发(evaporation · 汗液和水分蒸发带走潜热)。夜间露宿失温的能量损失分布大致是:传导(地面)40-50% · 对流(风)25-30% · 辐射(向天空)15-20% · 蒸发(湿衣)10-15%——所以地垫(切断传导)+ 挡风(切断对流)+ 反光保温毯(切断辐射)这三件事必须叠加。湿衣立刻换是切断蒸发损失的唯一办法,不能换就把湿衣拧干尽量贴身,体温倒过来烘干(下一节睡袋系统会展开)。

5.4睡眠系统三层叠加——地垫 · 睡袋 · 防水

睡眠系统不是「买一个 -10°C 的睡袋就够了」——它是地垫 + 睡袋 + 防水三层叠加的整体性能。地垫被严重低估、睡袋温标被严重误读、防水被完全忽视——三件事都做对的人很少。这一节讲清楚这三层每一层的最低门槛。

第一层:地垫 R 值——比睡袋更被低估

R 值(R-value)是热阻系数,描述材料阻挡热传导的能力——数字越大越保暖。这是地垫上的关键标号,但中国户外市场长期忽视它。一个无地垫的睡袋,无论温标多低,体温都会通过地面持续流失——地面温度通常比气温低 5-10°C,无隔绝的直接接触相当于把整张睡袋打对折。

第二层:睡袋温标——只看舒适不看极限

睡袋温标是新手最容易被骗的地方。国际标准 EN 13537 / ISO 23537 给一个睡袋三个温度数字:

  • 舒适温度(comfort temp)——标准女性体型在该温度可舒适入睡。这是你应该看的数字。
  • 极限温度(lower limit / limit temp)——标准男性体型蜷缩可勉强睡 8 小时不被冻醒,但不舒适。
  • 极端温度(extreme temp)——可在该温度下生存 6 小时不致死,但会冷伤;不是「可睡温度」,是「不至于死的最低线」。

舒适和极限通常差 8-10°C——比如某睡袋标「舒适 -2°C,极限 -10°C」,意思是普通男性能在 -2°C 舒适睡眠,在 -10°C 勉强熬一夜但要发抖,在 -28°C(极端温度)能活但会冻伤。新手最常见的死法是看着「-10°C 极限」以为这睡袋能在 -10°C 睡觉——这是用极限温度做计划的典型错误,实际睡到那个温度只会冻得醒一夜。

反共识 / 看舒适温标 + 再降 5°C 余量,不是看极限温标

选睡袋的正确规则:预期夜间最低温 = 舒适温度 - 5°C 余量。换句话说,要在 -5°C 夜里睡,买一个舒适温度 0°C 的睡袋(不是极限 -5°C 的)。新手最常买的「-10°C 极限」睡袋,实际舒适温度是 0 到 -2°C,只适合 +3 到 +5°C 的夜晚——和袋子标签上印的 「-10°C」 差着将近 15°C。这是中国户外市场长期的标签游戏,品牌喜欢印极限温度(数字看上去更猛)而不是舒适温度。买睡袋前找产品技术页核对 ISO 13537 测试数据里的 comfort temp 那一栏——这是唯一能信的数字。

第三层:防水——睡袋潮了等于报废

羽绒睡袋潮湿后保暖效率下降 60-80%,化纤睡袋下降 20-40%——这就是为什么羽绒优秀但怕湿、化纤迟钝但耐湿的本质区别。防水有三层:

  • 外层防水:睡袋自带的 DWR(durable water repellent)外壳处理 · 6-12 个月使用后需补涂
  • 压缩袋防水:防水压缩袋(dry sack)收纳,背包进水时睡袋是最后一道不能湿的物资
  • 使用中防水:不要让睡袋直接接触帐底(冷凝水)、不要钻进去时身上湿、不要嘴对着内层呼气(湿气积累)
操作 / 睡袋潮了怎么救——用身体烘干法

多日穿越里睡袋潮湿是常见事(冷凝水 / 雨水侵入 / 大汗渗透)。新手直觉做法是「明天太阳出来晒干」——这是错的,因为今夜你就要靠它过夜。

  1. 潮湿睡袋拧不出水的程度——继续穿,湿冷感会消失约 20 分钟(身体把贴身那层烘干)
  2. 能拧出水的程度——脱光内衣裤,只穿一层干的内层 + 套进湿睡袋,体温从内向外蒸发,大约 30-45 分钟内层会基本干
  3. 整袋湿透——这种情况下睡袋已经报废 8 成性能,优先策略是:① 穿上所有干衣,② 双人合睡叠加体温,③ 把湿睡袋当外层防风层而不是核心保暖层
  4. 羽绒湿透极端情况——用手抖松湿羽绒团,尽量恢复蓬松度(羽绒湿后会结块,完全不保暖);但 90% 情况下这一晚已经救不回来,优先考虑紧急避险下撤

原理是水的蒸发潜热:1g 水蒸发吸收 540 cal。湿睡袋在身体里烘干等于身体替睡袋耗了这部分能量——不舒服,但比湿冷睡眠安全得多。

5.5失温——为什么湿冷午后比干冷夜晚更致命

失温(hypothermia)是户外死亡的第一大死因,但它最反直觉的一点是:致死率最高的不是极冷。死亡发生最密集的温度区间不在 -20°C,而在 5 到 10°C——配上雨 + 风。理解这条悖论是这一章最重要的认知。

湿冷悖论——为什么 5°C 比 -10°C 更危险

原因不在物理,在心理 + 生理的复合作用:

  1. 心理:不警觉——5-10°C 是城市生活的春秋日间温度,人不会主动加衣、不会减速、不会停下评估状态。在 -20°C 干冷下,任何正常人都会本能加厚装备、提高警惕、缩短行程。
  2. 生理:湿衣 + 风冷叠加——风冷效应(wind chill)在 6°C 配 6 级风(13 m/s)时等效温度 -3°C;再叠加内衣湿透,体表蒸发损失放大 5-25 倍。结果是 5-10°C 的山里实际等效体感 -10 到 -15°C。
  3. 不下大雨,持续小雨——大雨人会主动停下找避雨,小雨人会「走快一点就到了」——结果几小时内冲锋衣 + 中层全湿透。
  4. 队友也不警觉——同伴看你「还能走」就以为没事;失温的第一阶段症状(寒颤、思维变慢、说话颠倒)很容易被误读为「累了」。

失温分级以核心温度(core temp,通常指食道或膀胱测温)为标尺,但野外没有这些设备,只能凭体征判断。

操作 / 中度失温的同伴体温伴入

这是野外最有效但被低估的救命动作——核心是用另一个人的核心体温替无法自主产热的失温者升温。

  1. 找避风点 + 厚地垫(切断地传导)+ 大睡袋或两个睡袋拼接
  2. 失温者剥下所有湿衣(湿衣留在身上失温反而加快)· 包裹干衣或直接进睡袋
  3. 同伴脱到内衣层 · 进入同一睡袋 · 紧贴失温者背部和胸前两人贴合
  4. 持续 30-60 分钟 · 期间观察呼吸 / 意识 / 寒颤恢复
  5. 意识恢复后逐步喂温糖水(每次小口 · 不超 50ml · 不能呛到)

规则:外部干热源(取暖器 / 火堆)对深度失温者反而有害——体表血管扩张让冷血回流核心,可能加重低温。同伴体温伴入是最缓慢但最安全的复温方式。

5.6营地厨房与排泄管理——熊罐 / 食物悬挂 / 70 米法则

营地选址决定夜间是否安全,营地组织决定夜间是否被打扰。打扰主要来自三个方向:食物气味招来野生动物、排泄物气味招来昆虫和小动物、人为废弃物长期改变营地周围生态。中国户外语境下,这三件事的优先级被严重低估——大多数人扎完帐就在帐边吃饭,留下残羹剩饭。这是夜间野猪 / 黑熊 / 流浪犬来访的最常见诱因。

70 米法则——厨房 / 帐篷 / 厕所三角形

NOLS 和北美 LNT(Leave No Trace 无痕户外)的标准:厨房、帐篷、厕所三点之间互相距离不少于 70 米(约 200 英尺),呈三角分布,且都距离水源 60 米以上。

  • 厨房 ↔ 帐篷 70m——隔开食物气味,避免野兽夜间被气味引到睡眠点
  • 厕所 ↔ 帐篷 70m——排泄物气味和卫生隔离
  • 厕所 ↔ 水源 60m+——避免粪便污染下游水源
  • 厨房 ↔ 水源 60m+——避免食物残渣和油脂进入水源

食物存储——熊罐与悬挂法

中国东北(长白山 / 大兴安岭)、川滇(贡嘎 / 四姑娘 / 香格里拉)、新疆部分山区都有黑熊或棕熊出没。熊的嗅觉是人类的 2000-3000 倍,营地里任何带气味的物品(食物 / 牙膏 / 防晒霜 / 卫生用品)都是诱饵。

  • 熊罐(bear canister)——硬壳塑料桶 · 熊撬不开;美国国家公园部分线路强制使用,中国川滇高山线路开始有人配。重量 1-1.5kg,价格 ¥800-1500。
  • 悬挂法(bear hang / PCT 法)——绳索悬挂食袋于树干之间,袋距地面 4 米以上、距树干 1.5 米以上,熊够不到。需要两棵合适距离的成年树,在密林是常用方案,在高山稀树带不可用。
  • 气味隔离袋(Opsak / 防熊袋)——双层密封袋,显著降低气味泄漏;不能替代熊罐或悬挂,但可以延缓野兽察觉时间。

排泄管理——6 英寸坑法则

大便处置的国际标准:cat hole 猫洞法——挖一个 6 英寸(15cm)深、4-6 英寸宽的坑,在坑里完成后用土覆盖填埋。这个深度是细菌降解最快、且不会被动物挖出来的平衡点。卫生纸两个选择:① 烧掉(干燥防火环境下,潮湿森林禁烧);② 装入封口袋带走(LNT 严格执行版,中国保护区核心区要求)。

注 / 中国语境下的熊罐

中国户外市场长期对「熊罐」这件事不重视——多数人觉得「中国又没熊」。这是错的:东北黑熊 / 棕熊数量在大兴安岭和长白山近年回升 30%(中国林草局 2022 数据),川滇黑熊一直稳定分布在贡嘎 / 四姑娘 / 香格里拉等热门线路沿线。2023 年青海年保玉则一队户外人夜间帐篷被棕熊撕开取食事件就是典型案例。不带熊罐的事实根源是中国户外文化里「野兽来了再说」的侥幸心理——但熊一旦把人的帐篷和食物建立联系,会持续到来,危险会指数级放大。在长白山 / 贡嘎 / 香格里拉这些有明确熊出没记录的线路上,熊罐应该被视作和净水器同级的必带装备。

5.7夜间野生动物来访的处置——光、声、不接近

在中国温带山地,夜间来访营地的主要是三种:黑熊(食物气味驱动)、野猪(食物气味 + 兽道驱动)、流浪犬(垃圾驱动,城市近郊高频)。其他偶发的有麂、獐子(无威胁)、豹猫 / 黄鼬(对小型物品好奇但不威胁人)。处置原则只有一条:不主动接近,用光和声让它知道这里有人然后让它自己离开。

三种主要来访者的处置

动物典型行为危险度处置
黑熊(中国东北 / 川滇)夜间嗅探营地 · 撕扯食物袋 · 通常避人中——但若有食物链建立则升高头灯爆闪 + 敲打金属锅(高频金属声 · 不是吼叫)· 集体站立显大 · 不要背对 · 不要跑;熊离开后立即转移营地

反共识 / 不要「装死」也不要「放食物」

电影和民间传说里有两个误区:① 「遇到熊装死」——这只对极少数情况(已经被攻击的灰熊)有效,对黑熊基本无效甚至有害;中国主要分布的是黑熊和野猪,装死等于让它继续靠近;② 「扔食物让它离开」——这是最危险的做法,因为它教会野兽「营地 = 食物」,下次它会更主动来访,而且会带它的同伴来。正确动作是让它明确感觉这里有人(光 + 声),然后不挡它的退路(给它一个明显的离开方向),让它自己走。野生动物的本能是避开冲突,只有被逼到角落或被建立食物关联后才会攻击人。

5.8失联场景——无装备一夜的庇护所

最坏情况:你和大队伍走散、装备丢了或没带、天色已晚来不及下撤、必须就地过夜。这种场景在中国户外事故里不罕见——2024 年北京户外救援队全年接警 280 起,其中「失联 + 露宿一夜」类型占 18%。这一节给一个最低限度的方案,假设你只有身上穿的衣服、手机、可能还有一个打火机。

四种最坏情况的庇护方案

  • 树洞 / 树根坑——林区夜晚找一棵大树(直径 ≥ 50cm)根部凹坑或树洞;铺一层厚树叶 / 干松针(15-20cm)阻断地传导;身体蜷缩进入;面朝树干背朝外(背对风);用衣物覆盖头颈(头颈是体表散热最快的部位之一)
  • 雪洞 / 雪坑(雪季)——积雪深 80cm 以上的环境,用手或登山杖挖一个能容下身体的坑,顶部覆盖树枝 + 雪块;入口朝下风;雪不冷,稳定的雪洞内温度可以维持在 -2 到 0°C 即使外界 -20°C;但挖洞时大汗是雪洞最大风险,要慢挖 + 频繁脱外层
  • 树枝堆(debris hut)——找一根倒木作主梁,沿主梁两侧铺斜杆(45° 角),再覆盖厚 50cm 以上的枯叶 / 松针;人钻入像钻入睡袋一样;这是 SAS 求生手册推荐的标准方案,适合温带林区
  • 同伴体温交换——如果有同伴在场,两人贴身蜷缩在背风处,共同用所有衣物覆盖;两人核心温度互助比一人保暖效率高 40-60%;条件允许时三人同蜷更好(中间人保暖最佳,轮换位置)
放在你身上 · 5°C 配小雨 / 队友说休息一下

剧本:你和三个同伴在秦岭东段一条主脊上,海拔 2800m。下午两点,气温 5-7°C,有零星小雨,风 5 级。你们已经走了 6 小时,前一个补给点过去 3 小时。最近一个同伴老张说「我有点累了,休息一下吧」,大家停下来吃东西。你注意到老张说话节奏快但内容颠倒了一次,把「今晚扎水窝子」说成「明早扎水窝子」。他冲锋衣下摆都是水印,内层抓绒看上去也湿了。

下面是你的时间轴处置——不要凭感觉,按顺序做。

时间状态判断具体动作
T+0(刚停下)同伴说话颠倒一次 · 内衣可能湿透不要假设「他只是累了」——在 5-7°C 湿冷环境下这是轻度失温初期信号;立刻停止前进
T+2 分钟评估周围找最近的天然背风物(大石 / 林缘 / 倒木侧)· 不超过 50 米直线
T+5 分钟组织避风点把背包靠墙摆成挡风墙 · 铺地垫或睡袋作隔地层 · 同伴脱湿外层 + 冲锋衣 + 中层 → 换干衣(借同伴的备用)
T+10 分钟启动产热给老张温糖水(保温杯里的热水 + 巧克力)· 让他做缓慢摆臂和深蹲(主动产热,不要剧烈)· 用空间毯 / 备用羽绒裹住
T+25 分钟评估恢复程度说话清晰度恢复 / 寒颤变明显(这是好兆头,寒颤说明身体还有能量产热)/ 主动表达「暖和一点了」 → 可考虑续走
T+30 分钟决策续走还是就地扎营恢复良好且距下个营地 < 2 小时 → 慢速续走;恢复不充分或距营地 > 2 小时 → 就地扎营;天色变暗 → 必须就地扎营;雨变大 → 必须就地扎营

最重要的一条:不要「再走一段看看」。湿冷午后失温的典型死亡路径就是「还能走一段、再走一段」,每走一段体温都掉一点,直到中度失温寒颤停止时已经救不回来。在 5-10°C 配雨配风的环境下,任何一个同伴出现思维迟缓或说话颠倒的初期信号,立刻停下处置——这是 2017 年鳌太事件最贵的一课。

注 / 高反夜睡的特殊问题

高海拔(> 3500m)夜间睡眠会出现特殊现象:周期性呼吸(Cheyne-Stokes respiration · 中枢性睡眠呼吸暂停),表现为 20-30 秒深呼吸后 10-20 秒完全停呼吸的循环,血氧饱和度在停呼吸期间可降到 70% 以下。这不是病,是身体对低氧环境的代偿,但带来浅眠 / 多梦 / 易醒 / 醒后头痛,严重时是急性高山病(AMS · acute mountain sickness)的初期表现。处置:① 不要熬夜或喝酒(都会加重),② 帐内保持通风(否则 CO2 累积),③ 睡袋温标比平地再降 5°C(高海拔夜温陡降),④ 若头痛持续 + 厌食 + 失眠 → 立即下降至少 500m 海拔,不要再继续上升。这是高海拔多日穿越夜里最容易被忽视的隐性风险。

「如果失联了」——本章升级版

如果故事走到了「失联且没装备需要露宿」,这一章的所有内容会以三种方式回到你身上:① 五避选址的逻辑你已经会用——即使没有装备,你也知道靠什么背风、远离什么、避开什么死木;② 湿冷悖论让你保持警觉——在 5-10°C 配雨的环境下你会主动找避风点而不是继续走;③ 四种最坏情况的庇护方案(树根坑 / 雪洞 / 树枝堆 / 同伴体温)给你具体的下一步动作,而不是茫然立在原地。最后一条:火,如果有火且能点起来——参考第四章——是你这一夜的最强武器,但选址五避不能因为有火就放松。
