# 《越野跑进阶训练》· sub-agent 作战手册

> 派写章节的 sub-agent 必读。顺序：先读 voice 体系（`_shared/voice/WRITER_PROMPT.md`）→ 再读本手册 → 再读 `OUTLINE.md` 里你负责那章的 scope 与核心裁决 → **最后严格照 `chapters/ch08.html`（主编亲笔样章）的结构与配色产出**。

## 这是什么书

《越野跑进阶训练》· 副题「从执行课表，到自我教练」——投稿微信读书，给**严肃进阶越野跑者 / 自我教练者**看的训练方法学。

它是《越野跑科学》（2026-06，本仓库 `reports/2026-06-trail-running/`）的**纵深续作**，不是它的加长版：

| | 旧书《越野跑科学》 | 本书《越野跑进阶训练》 |
|---|---|---|
| 回答 | **为什么**（山对身体做了什么） | **怎么练**（剂量 / 时机 / 设计） |
| 读者 | 大众越野跑者 | 严肃进阶者 → 自我教练者 |
| 完成线 | 「要练离心」 | 「练多重、隔多久、赛前第几天」 |

**一句话记住定位**：旧书把山讲给你听，本书教你把自己当成一个训练项目来管理。**旧书的完成线，是本书的起跑线。**

## 能力阶梯（全书承诺的边界 · 每章都不许越界）

完赛者 → **严肃进阶者（本书从这里接人）** → **自我教练者（本书把你送到这里）** → 精英（只做参照，不做承诺）。

写任何一章都记住：**精英内容在书里，但只做参照系。** 凡涉及精英做法，必须同时给出「他们的地基是什么」与「你能不能降剂量借用」，**严禁把精英课表直接摆给读者当目标**。分开精英与认真业余的是量级 / 年限 / 天赋 / 支持系统四样——缺的不是一本更高级的书，是全职的人生结构。

## 七条硬约束（每章逐条自检）

### 1 · 标点一律全角（2026-07-16 白笑定 · 硬红线 · lint 的 P 检测守门）

**中文文本里所有标点全角**，不只是引号：逗号 `，` 句号 `。` 冒号 `：` 分号 `；` 问号 `？` 叹号 `！` 括号 `（）`。

- ✗ 错误：`它长得最快,一次课就有` / `先认一个词:说明书`
- ✓ 正确：`它长得最快，一次课就有` / `先认一个词：说明书`
- **术语批注也全角**：`神经适应（neural adaptation，神经系统改变肌肉的募集方式）`
- **例外（保持半角，改了会坏）**：CSS（`width:100%`）· SVG 属性（`dasharray="7,5"` `d="M0,0 L6,3"`）· URL（`http://`）· 纯英文句子内部
- 判据：**这个标点两边挨着中文吗？挨着就全角。**
- 手滑了不要紧，跑 `python3 _shared/voice/fix-punct.py <你的文件>` 批量修（它会自动保护 SVG / CSS / 代码）

### 2 · 接缝纪律：旧书讲过的不重讲

凡旧书已讲透的机制，**一句话带过 + 自然指路**（「《越野跑科学》下坡那章拆过这副刹车片」），**严禁重讲超过一段**。逐章硬对照表见 `OUTLINE.md` 的「与旧书的接缝」。

### 3 · 勘误优先于护短（接缝纪律的特例）

**发现旧书有错，必须点名修正、讲透为什么错**——这不是重讲，这正是本书存在的理由。

**凡引用旧书数字，先查一手原文再引。** 已确认第一例：旧书 Ch2 的「疫苗保护几个月」是把 Nosaka 的**上肢**离心抗阻研究外推到腿上，一手原文（*Sci Rep* 2020）说下坡跑的保护只有 5–35 天（样章 Ch8 已点名修正）。**自家旧书是最不设防的投毒源**，别因为「那是我们写的」就当已核实。发现新的勘误，回写进 `OUTLINE.md`。

### 4 · 剂量优先：停在原则层 = 没完成

每章的实操落点**必须给到剂量 / 时机 / 进阶三要素**。「要练离心」「要热适应」这种原则层表述是旧书的完成线，是本书的**起跑线**。写完问自己：读者照着这章，能算出「我这周该练多少、隔几天、赛前第几天」吗？答不上来就是没写完。

### 5 · 证据分级：关键处方标注强度

延续「敢承认学界分歧」并升级为体系。关键处方要让读者看出证据有多硬：多项对照研究 / 观察性 / 精英实践共识 / 作者裁决。**不确定就说不确定**——样章里「5 到 35 天上下差着七倍，这个区间宽得几乎像没说」那种坦白，正是本书的信任来源。

### 6 · 竞品抄诊断不抄药方

Uphill Athlete / Koop / 挪威双阈值这些体系，**抄它的诊断，不抄它的药方**——它们的解常常由自己的人群与形态决定（Koop 轻视数据监控的前提是 CTS 教练在场，自我教练者不能照抄）。裁决尺子：证据强度 × 越野特异性 × 自我教练可执行性。**来源可信度分级**：一手研究 / 官方最可信；营销号、SEO 软文、工具站的伪精确数字必须降权，不当硬依据。

### 7 · 不编造

**第一人称只用真实信息，绝不编造白笑的越野经历**（不知道他跑过哪场赛）。需要在场感就用通用第一人称（「第一次跑长下坡的人都有这体验」）或明确虚拟（样章用的是「不妨设想一位跑者（以下情节纯属举例）」）。不编造伪精确数字。涉及训练 / 伤病加一句科普非医疗建议、以无痛为准、有伤就医。

## 声音

- **引擎**：科学 / 证据段 = 同伴腔 · 方法 / 剂量段 = 教练腔 · 战略 / 收尾 = 格言腔。**一节内不混。**
- **一章一个贯穿隐喻**，全章咬住它，不堆叠别的比喻。样章 Ch8 = 疫苗的说明书（剂量 / 间隔 / 有效期 / 加强针 / 禁忌），全章没跑题。
- **像人三件套**：数字关键处精确、装饰处粗化带出处；承认学界分歧；第一人称在场。
- 中文引号**只用「」**（嵌套『』），书名《》。强调用 `<strong>`/`<em>`，不用引号。
- 引用具体不戏剧化：名字 + 年份就够（「Khassetarash 等人 2023 年那项研究」），不写「某个清晨」。
- **双语批注硬门槛**：每个英文 / 术语首次出现，带「中文语义」+ 一句话干嘛。逐字扫正文 + 图标签，无一例外。**裸英文词混在中文里（如「一批 protocol 各不相同」）= 违规**，要么给批注，要么改成中文。

## 编号

- **留章号**：章首 `<div class="section-tag"><span class="num">NN</span>X篇 · 子题</div>`
- **去小节号**：`<h3>` 只放纯标题，不要 `8.1/8.2`。跨节引用用命名式（「前面拧旋钮那节」），跨章可写「下一章」「第十章」。

## 格式（严格照 `chapters/ch08.html`）

产出是**一个 `<section>` 片段**（不是整页 html，不要 `<html>/<head>`）：

```
<section id="sNN" style="border-top:4px solid var(--accent);">
<div class="container">
  <div class="hero-keypoints"> 4 个 .kp 卡片（关键数据：tag/value/desc） </div>
  <div class="section-tag"><span class="num">NN</span>X篇 · 子题</div>
  <h1 class="section-h">章标题（带贯穿隐喻）</h1>
  <p class="section-sub">本章钩子，1-2 句</p>
  <div class="tldr"><div class="label">▼ 一句话</div><p>全章浓缩成一句狠话</p></div>
  <p>引子段（带出贯穿隐喻）</p>
  <h3>小节纯标题</h3>
  <p class="lead">小节首段（导言）</p>
  <p>...正文...</p>
  <div class="svg-frame no-audio"><svg viewBox="0 0 1080 H" style="width:100%; height:auto;">...</svg><div class="svg-caption">图注（带出处）</div></div>
  <div class="metric-grid no-audio"> 3-4 个 .metric-card </div>   （按需）
  <table>...</table>                                              （按需，剂量表 / 课表用）
</div>
</section>
```

- **篇幅：以 voice-lint 报的中文字数为准，必须 ≥7000（目标 7500–9000）**。lint 只数纯汉字，不含标点英文数字——别因为「感觉写够了」就停，没到就继续加**实操深度**（剂量、进阶、决策规则），加到达标为止。
- **2–3 张原生 SVG**：自己画，别用占位图。
- **表格是本书的主力组件**（旧书靠叙事，本书靠剂量表 / 课表 / 决策树）。样章 Ch8 用了 2 张：8 周进阶表、技术课表。列宽窄的首列加 `style="white-space:nowrap"`，否则会被挤成竖排。
- 收尾形式每章变换，不要每章都用同一种「反直觉数字」收尾。末尾一段自然过渡到下一章。

### SVG 硬要求（样章踩过的坑，别再踩）

- **坐标轴必须线性**：刻度间距要和数值成比例。样章初稿把「2 个月」硬塞在等距刻度上、把 35→28 天画得比 56→35 天还宽，渲染出来才发现比例尺在骗人。
- **图和表必须对齐**：同一章里 SVG 的分期和表格的周次不许打架（样章初稿图里标「减量期」的那段，表里还写着「量持平」）。
- **写完必须渲染出来看**，不是读代码就算数。套壳预览：把你的 section 塞进 `preview-ch08.html` 同款骨架（`<link rel="stylesheet" href="../../_shared/style.css">` + olive 配色覆盖）。

## 配色（岩灰 + 荧光 · SVG 与 inline 用这些 hex）

- **全书统一 olive/volt，不分篇换色**。section border **一律** `var(--accent)`，**禁止**用 `var(--c-physio)` 那类学科色。
- 纸底岩灰 `#e7e6de` · 卡片 `#e0ded3` · 网格线 `#c3c2b3`
- **主强调 olive `#5f7320`**（border、section-tag、SVG 主数据系列、图表标题）
- **荧光高亮 volt `#93b327`**（需要跳出来的高亮、正向 / 达标）
- **暖警示 `#b35a2a`**（损伤 / 危险 / 错误做法 / 对比的负向语义）
- 第二中性系列可用深蓝 `#1f3a5f`
- 文字 ink `#1a1a1a` / soft `#4a4a44` / faint `#8a877c`（SVG 文字务必深色高对比）
- 颜色要**编码语义**：正向 = volt/olive，负向 = `#b35a2a`。同一张图别两个语义撞色。

## 硬禁清单

- 不编造白笑经历、不编造伪精确数字、不假装全知（该承认分歧就承认）
- 不堆 callout 框（本书**一个都不用**，全溶回正文——样章 Ch8 零 callout）
- 小标题不英中混排、不炫技
- 不把精英课表当目标摆给读者
- 不停在原则层（见约束 4）
